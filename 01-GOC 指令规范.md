
# 📄 GOC 在线编程语言 — 指令规范 PRD（标准版）

---

# 一、文档目标

本规范用于定义 **GOC（Graphical Online Coding）语言的完整指令体系与语法规则**，用于：

* 在线编程平台解析执行
* AI 自动生成代码的标准约束
* 教学与文档统一参考

---

# 二、语言基本约定

## 2.1 核心对象

GOC 仅提供一个全局绘图对象：

```cpp
pen
```

所有绘图相关操作必须通过 `pen` 调用。

---

## 2.2 调用规范（强制）

### ✅ 链式调用

```cpp
pen.size(2).c(1).fd(100).rt(90);
```

### ✅ 多行链式（推荐）

```cpp
pen
  .size(2)
  .c(1)
  .fd(100)
  .rt(90);
```

---

## 2.3 语法规则

| 规则               | 说明 |
| ---------------- | -- |
| 每条语句必须以 `;` 结尾   | 强制 |
| 所有方法必须挂在 `pen` 上 | 强制 |
| 支持链式调用           | 强制 |
| 支持 C++ 风格控制流     | 支持 |

---

# 三、数据类型

支持基础 C++ 子集：

```cpp
int a = 10;
double b = 3.14;
string s = "hello";
```

---

# 四、控制结构

## 4.1 条件语句

```cpp
if(a > 10){
    pen.fd(100);
}else{
    pen.bk(50);
}
```

---

## 4.2 循环语句

```cpp
for(int i=0;i<4;i++){
    pen.fd(100).rt(90);
}
```

# 五、绘图指令（核心）

---

## 5.1 画笔控制

| 指令           | 参数        | 说明        |
| ------------ | --------- | --------- |
| pen.up()     | 无         | 抬笔（移动不绘制） |
| pen.down()   | 无         | 落笔（开始绘制）  |
| pen.size(n)  | n:number  | 设置线条粗细    |
| pen.c(color) | color:int | 设置颜色      |

---

## 5.2 移动与方向

| 指令              | 参数           | 说明        |
| --------------- | ------------ | --------- |
| pen.fd(n)       | n:number     | 前进        |
| pen.bk(n)       | n:number     | 后退        |
| pen.rt(angle)   | angle:number | 右转        |
| pen.lt(angle)   | angle:number | 左转        |
| pen.moveTo(x,y) | x,y:number   | 移动到指定位置   |
| pen.lineTo(x,y) | x,y:number   | 绘制直线到指定位置 |

---

## 5.3 图形绘制

| 指令          | 参数         | 说明     |
| ----------- | ---------- | ------ |
| pen.o(r)    | r:number   | 绘制圆    |
| pen.oo(r)   | r:number   | 绘制实心圆  |
| pen.e(a,b)  | a,b:number | 绘制椭圆   |
| pen.ee(a,b) | a,b:number | 绘制实心椭圆 |
| pen.r(w,h)  | w,h:number | 绘制矩形   |
| pen.rr(w,h) | w,h:number | 绘制实心矩形 |

---

## 5.4 文本与图片

| 指令                | 参数                    | 说明       |
| ----------------- | --------------------- | -------- |
| pen.text(x,y,str) | x,y:number；str:string | 绘制文字     |
| pen.pic(url)      | url:string            | 显示图片     |
| pen.picL(x,y,url) | x,y:number；url:string | 指定位置显示图片 |

---

## 5.5 显示控制

| 指令         | 参数 | 说明 |
| ---------- | -- | -- |
| pen.show() | 无  | 显示 |
| pen.hide() | 无  | 隐藏 |

---

## 5.6 动画与控制

| 指令           | 参数       | 说明       |
| ------------ | -------- | -------- |
| pen.speed(n) | n:number | 设置执行速度   |
| pen.wait(t)  | t:number | 等待（毫秒）   |
| pen.pause()  | 无        | 暂停执行     |
| pen.ctime()  | 无        | 获取或触发计时器 |

---

## 5.7 音频控制

| 指令           | 参数 | 说明      |
| ------------ | -- | ------- |
| pen.sound()  | 无  | 播放声音    |
| pen.soundL() | 无  | 控制或加载声音 |

# 八、合法示例

---

# 九、错误定义（规范层）

---

## 9.1 语法错误

| 类型       | 示例          |
| -------- | ----------- |
| 缺少分号     | pen.fd(100) |

## 9.2 参数错误

| 类型   | 示例          |
| ---- | ----------- |
| 类型错误 | pen.fd("a") |
| 参数缺失 | pen.rt()    |

---

## 9.3 非法指令

```cpp
pen.xxx(); // 未定义方法
```

# 十、兼容性说明

## 10.1 不支持旧语法

以下写法不合法：

```cpp
pen;
.fd(100);
```

---

## 10.2 必须改为

```cpp
pen.fd(100);
```

---

# 十一、语言边界

GOC 规范仅包含：

* 绘图 API
* 基础控制流
* 简单变量系统

不包含：

* 函数定义
* 类
* 模块系统
* 异步机制

---

# 十二、总结（规范定义）

GOC 指令体系定义如下：

```text
统一入口：pen
统一风格：链式调用
控制结构：C++ 子集
执行单位：语句（;）
```
