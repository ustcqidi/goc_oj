# 一、产品概述

## 1.1 产品定位

一个面向初学者与教学场景的**在线图形化编程学习平台（Web GOC）**，融合代码编写与可视化绘图执行，降低编程学习门槛。

## 1.2 目标用户

* 编程初学者（中小学 / 非科班）
* 编程教学老师
* 自学用户（偏兴趣驱动）
* 图形化编程入门用户

## 1.3 核心价值

* **所见即所得**：代码执行结果实时体现在绘图区
* **降低学习门槛**：封装命令（如 `.fd()`、`.rt()`）替代复杂语法
* **教学友好**：命令面板 + 示例驱动学习
* **交互式体验**：输入输出 + 图形 +代码三位一体

# 三、功能模块设计

## 3.1 顶部控制区（主菜单）

### 模块组成：

* 清屏按钮
* 画坐标按钮
* 运行按钮
* 命令快捷入口
* 倍速控制（1.0倍）

### 功能说明：

| 功能   | 描述            |
| ---- | ------------- |
| 清屏   | 清空绘图区域 + 重置状态 |
| 画坐标  | 在绘图区绘制参考坐标系   |
| 运行   | 执行当前代码        |
| 命令入口 | 快速插入代码模板（推测）  |
| 倍速控制 | 控制绘图执行动画速度    |

### 用户意图：

* 快速验证代码
* 控制执行环境
* 降低调试成本

---

## 3.2 左侧绘图区（核心可视化区域）

### 功能说明：

* 显示代码执行后的图形结果
* 支持路径绘制（类似 Turtle）
* 支持图形绘制（圆、矩形、图片等）

### 推测能力：

* 实时绘制（逐步执行）
* 支持动画（结合 speed）
* 支持状态（笔位置 / 朝向）

### 用户意图：

* 通过视觉反馈理解代码逻辑
* 强化空间认知与程序执行过程

---

## 3.3 右侧代码编辑区（编程核心）

### 功能说明：

* 支持 C++ 风格语法
* 支持输出（cout）
* 支持函数调用（p.fd / p.rt 等）
* 基础代码结构（main函数）

### 推测能力：

* 语法高亮
* 自动补全（部分）
* 错误提示（未展示）

### 用户意图：

* 编写程序逻辑
* 控制绘图行为
* 学习编程语法

---

## 3.4 底部数据区（输入输出）

### 模块组成：

* cinWin（输入）
* coutWin（输出）

### 功能说明：

| 模块      | 描述     |
| ------- | ------ |
| cinWin  | 用户输入参数 |
| coutWin | 程序输出结果 |

### 用户意图：

* 模拟标准输入输出
* 支持交互式程序

---

## 3.5 命令参考区（教学辅助核心）

### 模块划分：

#### 1️⃣ 绘图命令区

* pen（画笔）
* fd（前进）
* rt（右转）
* lt（左转）
* c（颜色）
* size（粗细）
* up/down（抬笔/落笔）
* moveTo（移动）
* lineTo（画线）
* text（文字）
* pic（图片）

#### 2️⃣ C++命令区

* int / double / string
* if / else
* for循环
* cin / cout

---

### 功能说明：

* 点击命令 → 插入代码（核心能力）
* 显示命令说明（学习辅助）

### 用户意图：

* 降低记忆成本
* 快速上手
* 边学边用

---

# 四、核心交互流程

## 4.1 基础使用流程

```
进入页面
  ↓
查看命令参考
  ↓
点击命令 or 手写代码
  ↓
点击【运行】
  ↓
绘图区展示结果
  ↓
查看输出区（cout）
```

---

## 4.2 输入驱动流程

```
用户输入 cinWin
  ↓
点击运行
  ↓
程序读取输入
  ↓
输出结果（coutWin）
```

---

## 4.3 调试流程（推测）

```
修改代码
  ↓
运行
  ↓
观察图形 / 输出
  ↓
重复调整
```

# 五、状态与异常处理

## 5.1 状态设计

| 状态   | 描述       |
| ---- | -------- |
| 初始状态 | 无绘图      |
| 执行中  | 正在绘制（动画） |
| 执行完成 | 图形渲染完成   |
| 输入等待 | 等待用户输入   |
| 错误状态 | 编译或运行错误  |

---

## 5.2 异常处理（推测）

### 代码错误

* 编译错误提示（语法错误）
* 高亮错误行

### 运行异常

* 无限循环（超时中断）
* 非法命令调用

### 输入异常

* 类型不匹配
* 缺失输入
