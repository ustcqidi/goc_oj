## 001 - 螺旋三角形

### 图形分析

红色螺旋三角形，顶点朝右。由一系列等差递增的线段依次连接构成，每画完一条边就右转 120°，边长从 0 开始每次增加 10，共 20 条边（0, 10, 20, … 190）。

### 核心指令

`for` 循环、`pen.fd()`、`pen.rt()`、`pen.c()`

### 考察点

- `for` 循环中用循环变量控制边长（`i * 10`）
- 正三角形的外角为 120°（转角知识）
- 等差数列的应用（每条边比上一条多 10）

### 数学对应

| 数学概念 | 对应关系 |
|---------|---------|
| 等差数列 | 边长 0, 10, 20…190，公差 10 |
| 正三角形外角 | 每次右转 120° |
| 周长累加 | 可引申为数列求和 |

### 参考 Solution

```cpp
pen.c(1);
for(int i = 0; i < 20; i++){
    pen.fd(i * 10).rt(120);
}
```

### 可扩展出题方向

| 变体 | 改动点 | 难度变化 |
|------|--------|---------|
| 螺旋正方形 | 转角改为 90° | 相近 |
| 螺旋六边形 | 转角改为 60° | 相近 |
| 等比螺旋 | 边长改为 `i * i` 或 `2^i` | 升高 |
| 颜色随圈变化 | 每圈用 `pen.c(i % 16)` | 升高 |
| 反向螺旋 | `lt` 替代 `rt` | 相近 |

## 004 - 放飞气球

### 图形分析

4个气球沿放射状路径均匀分布，从中心出发：画笔抬起，左转60°后，循环4次，每次向外前进350步（放一个气球）再后退350步，然后右转 120/3 = 40°。形成以中心为原点的4方向放射分布。

### 核心指令

`for` 循环、`pen.up()`、`pen.fd()`、`pen.bk()`、`pen.lt()`、`pen.rt()`

（原题还使用了 `pen.picL`/`pen.pic`/`pen.picU` 扩展命令，与 OJ 不兼容）

### 考察点

- 抬笔 `pen.up()` + 来回移动（`fd` + `bk`）实现放射状图案
- 均匀分角：整圆 360° ÷ 4 = 90°（此题用 120/3=40° 配合初始 lt(60)，等效均匀）
- 循环变量不影响路径长度（固定距离）

### 数学对应

| 数学概念 | 对应关系 |
|---------|---------|
| 圆的等分 | 4 个方向，旋转步长 120/3=40° |
| 来回路径 | fd(350).bk(350) 实现点状放置 |
| 初始偏转 | lt(60) 调整起始方向 |

### 参考 Solution

```cpp
// 标准 GOC 等价版（不使用扩展图片命令，改用 pen.o 画点表示气球位置）
pen.up();
pen.lt(60);
for(int i = 0; i < 4; i++){
    pen.fd(100).o(10).bk(100).rt(40);
}
```

### 可扩展出题方向

| 变体 | 改动点 | 难度变化 |
|------|--------|---------|
| 6方向放射 | 循环次数改 6，转角 60° | 相近 |
| 渐变距离放射 | fd(i*50) 控制每臂长度不同 | 升高 |
| 彩色放射线 | 每臂用不同 pen.c(i) | 相近 |

### 参考 Solution

## 010 - 旋转图形

### 图形分析

n 折旋转对称图案：从中心出发，循环 n 次，每次前进 100 步后退 100 步，右转 360/n°。形成 n 条等角分布的放射臂（类似旋转对称的雪花或星形）。结合条件分支根据 n 值改变显示内容。

### 核心指令

`for` 循环、`if` 条件、`pen.fd()`、`pen.bk()`、`pen.rt(360.0/n)`

### 考察点

- 等分圆：转角 = 360° ÷ n
- `n` 作为变量控制旋转次数和转角（双重参数化）
- `if` 条件判断选择不同行为
- 浮点除法：`360.0/n` 而非 `360/n`

### 数学对应

| 数学概念 | 对应关系 |
|---------|---------|
| 旋转对称 | n 折对称图案 |
| 圆等分 | 转角 360°/n |
| 浮点精度 | 360.0/n 保证小数精度 |

### 参考 Solution

```cpp
// 标准GOC等价版（n固定为6，用o标记顶点）
int n = 6;
for(int i = 0; i < n; i++){
    pen.fd(100).o(8).bk(100).rt(360.0/n);
}
```

### 可扩展出题方向

| 变体 | 改动点 | 难度变化 |
|------|--------|---------|
| 固定n的放射线（标准OJ可用） | n取固定值4/5/6/8 | 入门~进阶 |
| 奇偶不同颜色 | if(i%2==0) pen.c(1) else pen.c(2) | 进阶 |
| 渐变臂长 | fd(i*20+50) 每臂不等长 | 进阶 |
| 旋转正n边形环 | 每臂末端画小正多边形 | 高级 |

## 014 - 挑糖果

### 图形分析

循环读入10对数据，用 if 过滤出满足条件（软糖）的项，每次满足条件就画一个彩色实心圆并前进60步，形成一排不规则间距的彩色圆。是「循环 + 条件过滤 + 绘图」三合一的综合题。

```cpp
pen.rt(90).up();

```

## 018 - 自定义颜色的迷宫

### 图形分析

方形螺旋迷宫：每条边比上一条长10步（等差数列），每次右转90°构成方形螺旋；颜色按奇偶交替（i%2==0 用11号紫色，i%2==1 用3号绿色）。共15条边，边长10,20,…,150。

此题与「螺旋四边形」类似，但增加了奇偶颜色变化，是两种考察点的叠加：等差数列 + 奇偶判断。

### 核心指令

`for` 循环、`if(i%2==0)`、`pen.c()`、`pen.fd(len)`、`pen.rt(90)`、`len = len + 10`

### 考察点

- 等差数列：len 从10开始，每步+10
- 奇偶判断控制颜色：i%2==0/1

## 094 - 山峰（2018南海区比赛）

### 图形分析

输入n个高度，画n个矩形（宽20）表示山脉，再第二遍遍历找山峰（两侧都低于它），用实心矩形覆盖标记。使用 `moveTo/r/rr` 绝对坐标绘图。

### 核心指令

`cinWin()`, 数组, `moveTo(x, y)`, `r(w, h)`, `rr(w, h)`, 条件判断

### 考察点

- 数组存储 + 两趟遍历
- 山峰条件：`arr[i-1]<arr[i] && arr[i]>arr[i+1]`
- `moveTo` 绝对定位，`rr` 覆盖实心标记

### 参考 Solution

```cpp
int arr[15];
int main() {
    cinWin();
    int n, m;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> m;
        p.moveTo(i * 20, m / 2.0).r(20, m);
        arr[i] = m;
    }
    for (int i = 2; i <= n - 1; i++) {
        m = arr[i];
        if (arr[i - 1] < m && m > arr[i + 1])
            p.moveTo(i * 20, m / 2.0).rr(20, m);
    }
    return 0;
}
```

> ⚠️ 使用 cinWin, moveTo, r(), rr() 扩展命令。
