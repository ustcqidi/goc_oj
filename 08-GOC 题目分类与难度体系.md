# GOC 题目分类与难度体系

本文档定义 GOC 题目的难度等级、知识点标签体系，以及与小学初中数学的对应关系，作为出题和选题的分类依据。

---

## 一、难度等级

### 三级定义

| 等级 | 英文值 | 中文 | 目标学生 | 核心特征 |
|------|--------|------|----------|----------|
| 入门 | `easy` | 入门 | 小学中高年级、刚接触编程 | 固定参数、单循环、直接观察规律 |
| 进阶 | `medium` | 进阶 | 初中、有一定编程基础 | 循环变量控制参数、多层逻辑、数学推导 |
| 高级 | `hard` | 高级 | 竞赛准备、初高中 | 条件分支、多图形组合、变量计算、坐标系 |

### 各难度详细说明

#### 入门（easy）
- 只使用 `fd` / `bk` / `rt` / `lt` / `c` / `size`
- 循环次数固定、参数不变（如 `pen.fd(100).rt(90)`）
- 或循环变量直接乘以固定系数（如 `pen.fd(i*10).rt(120)`）
- 不需要计算角度，角度直接给出
- 数学：数数、简单乘法、直接观察

#### 进阶（medium）
- 需要用「外角 = 360° / 边数」推导转角
- 循环变量同时控制多个参数（边长 + 颜色同时变化）
- 可能用到 `up()` / `down()` / `moveTo` 定位
- 图形有一定视觉复杂度（多圈、多形叠加）
- 数学：等差数列、正多边形角度、对称

#### 高级（hard）
- 含 `if/else` 判断（如奇偶颜色交替、条件变向）
- 需要多个变量配合（外层变量 × 内层变量）
- 涉及坐标系（`moveTo` / `lineTo`）、绝对定位
- 可能有双重循环
- 数学：规律总结、坐标几何、稍复杂的数列

## 二、知识点标签体系

### 2.1 指令类标签（直接对应 01 指令）

| 标签 | 对应指令 | 说明 |
|------|----------|------|
| `循环` | `for` | 用循环重复绘制 |
| `条件` | `if/else` | 用条件控制不同效果 |
| `前进后退` | `fd` / `bk` | 基础移动 |
| `转向` | `rt` / `lt` | 方向控制 |
| `抬笔落笔` | `up` / `down` | 不连续绘制 |
| `颜色` | `c(n)` | 颜色控制 |
| `线宽` | `size(n)` | 粗细控制 |
| `坐标移动` | `moveTo` / `lineTo` | 绝对坐标绘制 |
| `圆形` | `o` / `oo` | 画圆、实心圆 |
| `矩形` | `r` / `rr` | 画矩形、实心矩形 |
| `变量计算` | 变量声明与运算 | 用变量控制参数 |

### 2.2 图形类标签

| 标签 | 说明 | 代表题型 |
|------|------|----------|
| `正多边形` | 正三角形、正方形、正五/六/八边形等 | 画正六边形、画五角星 |
| `螺旋图案` | 等差或等比递增边长形成的螺旋 | 螺旋三角形、螺旋正方形 |
| `放射线` | 从同一点出发的多条射线 | 放射线图案 |
| `同心图形` | 大小递增的同一图形层叠 | 同心圆、同心正方形 |
| `网格` | 横竖等间距直线 | 方格网 |
| `多边形环` | 多个相同图形绕中心旋转排列 | 花瓣、风车 |

### 2.3 数学类标签

| 标签 | 数学概念 | 对应 GOC 场景 |
|------|----------|--------------|
| `等差数列` | 公差固定的数列 | 边长每次 +n（螺旋系列） |
| `角度` | 转角、外角、内角 | rt/lt 参数 |
| `正多边形角度` | 外角 = 360°/边数 | rt(360/n)，n 边形 |
| `对称` | 轴对称、中心对称 | 放射图、风车 |
| `倍数关系` | n 的倍数、余数 | `i*n`、`i%k` |
| `坐标` | 平面直角坐标系 | moveTo(x,y) |

## 三、与小学初中数学的对应

| 数学主题 | 年级参考 | 对应 GOC 题型 | 建议标签 |
|----------|----------|---------------|----------|
| 乘法口诀、简单倍数 | 小学 3-4 年级 | 画等边三角形（3×120=360） | `循环` `角度` |
| 正多边形的角 | 小学 5-6 年级 | 画正多边形（外角=360°/n） | `正多边形角度` |
| 等差数列初步 | 小学 5-6 年级 | 螺旋系列（边长 0,10,20…） | `等差数列` `螺旋图案` |
| 周长与面积 | 小学 5-6 年级 | 同心图形（周长递增） | `同心图形` |
| 轴对称与中心对称 | 初中 7 年级 | 放射线、风车、花瓣 | `对称` |
| 简单方程与变量 | 初中 7 年级 | 含变量的 if 分支题 | `变量计算` `条件` |
| 平面直角坐标系 | 初中 7 年级 | 网格图、坐标绘制 | `坐标` `网格` |
| 整除、余数 | 小学 5-6 / 初中 | 颜色按奇偶交替（i%2） | `条件` `倍数关系` |

## 四、题目分布建议

批量出题时的推荐分布（以 15 道题为例）：

| 难度 | 数量 | 说明 |
|------|------|------|
| 入门（easy） | 7-8 道 | 覆盖基础图形：正多边形、螺旋、放射 |
| 进阶（medium） | 4-5 道 | 覆盖多圈、组合图形、颜色变化 |
| 高级（hard） | 2-3 道 | 含条件判断或双重循环 |

标签覆盖建议：每批题目需包含 `循环`、`正多边形角度`、`等差数列`、`颜色`、`螺旋图案`、`放射线`、`同心图形` 等主要标签。
