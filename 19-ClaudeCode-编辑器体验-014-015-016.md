## 三、GOC-014：语法高亮

- **目标**：GOC 关键字（如 for、if、int）、pen 方法（fd、rt、c 等）、字符串、注释等用不同颜色/样式区分。
- **实现**：若当前为 textarea，可用 overlay 或 contenteditable+span 实现简单高亮；或引入轻量高亮库。关键字与 API 列表来自 01。统一配色，不刺眼。
- **验收**：输入含 for、pen.fd、字符串的代码，能明显区分颜色。

---

## 四、GOC-015：自动补全

- **目标**：输入 `pen.` 或 Ctrl+Space 时弹出候选列表（fd、rt、lt、c、size 等），选中后插入。
- **实现**：需 GOC/pen API 列表（01）；若为 textarea，可监听输入与按键，在光标附近显示浮层列表并处理选中；若接入 Monaco 则用其 API。
- **验收**：输入 `pen.` 后出现候选，选择一项能插入到光标处。

---

## 五、GOC-016：代码格式化

- **目标**：一键整理缩进与换行，如统一 4 空格缩进、大括号换行等。
- **实现**：简单规则即可（缩进、换行），前端 JS 处理当前编辑器内容后写回。
- **验收**：乱缩进的代码点击「格式化」后缩进与换行一致。
