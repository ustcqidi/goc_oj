## 三、GOC-027：AI 生成参考代码

- **目标**：用户点击「生成参考」后，根据当前题目描述生成参考代码，仅供对照、不强制；带免责提示。
- **实现**：前端将当前题目 title + description 作为 prompt 上下文，调用外部 API（或占位端点）；prompt 中明确约束：仅使用 01 指令、符合 09 风格。返回的代码展示在可折叠区域或二次确认后插入编辑器。API 端点可配置（如环境变量或配置项）。
- **验收**：点击生成后能拿到一段 GOC 代码并展示；有免责文案。

---

## 四、GOC-028：AI 解释选中代码

- **目标**：用户选中一段代码后点击「解释」，展示这段代码在做什么（自然语言）。
- **实现**：前端将选中文本作为 prompt 上下文，调用 API 返回解释并展示（如弹层或侧栏）。API 可与 027 共用或单独配置。
- **验收**：选中含 pen.fd、for 等的代码，点「解释」能看到自然语言说明。
