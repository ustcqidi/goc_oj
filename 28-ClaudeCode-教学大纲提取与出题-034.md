## 实现要点

### 一、大纲要点摘要

1. **若 PDF 文本可提取**  
   - 逐页或按章节提取：学段（小学/初中）、年级、模块/章、节或主题、核心概念与能力要求。  
   - 写成结构化文档，例如：  
     - 一级：学段  
     - 二级：年级  
     - 三级：模块（如数与代数、图形与几何、统计与概率等）  
     - 四级：具体知识点或能力描述  

2. **若 PDF 为扫描/图像版**  
   - 当前仓库中的 PDF 若通过 read_file 得到多为空白，则视为图像版。  
   - 可采取：  
     - **方案 A**：根据「义务教育数学教学大纲」的通用结构，写一份**通用大纲要点模板**（常见学段、年级、模块、典型知识点），并在文档中注明「需结合实际 PDF 或教材补充具体表述」。  
     - **方案 B**：若你有 OCR 或外部文本版大纲，将内容粘贴到 `goc/docs/义务教育数学教学大纲-要点摘要.md`，再让 Claude 做结构化整理与映射。  
   - 本文档默认：**先产出 goc/docs/义务教育数学教学大纲-要点摘要.md**，内容为「结构 + 能写出的要点」；若部分无法从 PDF 直接得到，在摘要中标注「待补充」或「参考课标/教材」。

3. **与 GOC 的衔接**  
   - 在摘要末尾或单独小节，标注**哪些知识点适合用 GOC 画图题呈现**，例如：轴对称、正多边形、圆与弧、坐标、规律与数列、比例等，与 11 中 CURRICULUM_MAP（正多边形、轴对称、圆与弧、坐标系、螺旋线、色彩规律、旋转）对应。

### 二、大纲→题目映射或新题

1. **映射表**  
   - 在摘要文档或单独文件中，列出「大纲模块/知识点 → 建议题目类型 / 现有题 id / 待出题」。  
   - 例如：小学四年级「轴对称」→ 可出对称图案题，对应 tags/curriculum「轴对称」；「圆的周长」→ 可出画圆、等分圆周题，对应「圆与弧」。

2. **新题（可选）**  
   - 若 033 未做或题量仍不足，可基于映射直接新增若干道题，符合 07 Schema，并写入 problems/、index.json，运行 sync；新题在 description 或 tags 中可引用「几年级、什么知识点」。  
   - 若 033 已做满约 150 题，本任务以「摘要 + 映射」为主，新题为可选补充。
