# GOC Online Judge

**图形化编程在线判题系统** — 写代码，看图形，自动判分。

🔗 **在线体验：https://ustcqidi.github.io/goc_oj/**

---

## 这是什么

GOC 是一种面向中小学生的图形化编程语言，语法是 C++ 子集，通过控制画笔绘图：

```cpp
// 用循环画正方形
for (int i = 0; i < 4; i++) {
    pen.fd(100).rt(90);
}
```

代码运行后，画布上实时出现图形。不是拖拽积木，是真正在写代码。

---

## 为什么做这个

市面上有两类产品：
- **Scratch / 编程猫**：图形化拖拽，有趣但零代码转移价值，学完对竞赛没帮助
- **洛谷 / 蓝桥杯OJ**：真正的 OJ，但对初学者太难，没有视觉反馈

GOC OJ 填的是中间这段空白：**图形反馈 + 真实代码语法 + 自动判题**，适合 Scratch 毕业生过渡，也适合信息学竞赛入门前的热身。

---

## 功能

- **233 道题**：入门 126 / 进阶 94 / 高级 13，难度连续覆盖
- **自动判题**：提交后对比执行轨迹，即时给出通过 / 未通过
- **实时绘图**：代码跑动画，看到画笔一步步走
- **断点调试**：单步执行，每步看画布状态
- **分层提示**：不会做可以逐步解锁提示，不直接给答案
- **参考解法**：通过后可查看标准解法
- **进度持久化**：做题记录、最佳用时、星标收藏保存在本地
- **深色模式 / 字体调节**

---

## 技术

纯前端，无需后端，可直接静态部署。

- 执行器：自研 GOC 解析器（JS），支持完整 GOC 指令规范
- 判题：对比执行轨迹 JSON，坐标容差 ±2px，角度 ±1°
- 题目：JSON 格式，可直接扩展

## 本地运行

```bash
# 克隆仓库
git clone git@github.com:ustcqidi/goc_oj.git
cd goc_oj

# 用任意静态服务器打开
npx serve .
# 访问 http://localhost:3000/oj/
```
