# 义务教育数学教学大纲 — 提取说明（供 GOC 出题用）

本文档说明如何从 **《义务教育数学教学大纲》**（`docs/义务教育数学教学大纲.pdf`）提取核心内容，用于结合 GOC 题目范式出题（产品想法 **GOC-034**）。

---

## 目标

- **输入**：`goc/docs/义务教育数学教学大纲.pdf`
- **产出**：
  1. **大纲要点摘要**：学段（小学/初中）、年级、模块/章节、核心概念与能力要求（可写在本目录下，如 `义务教育数学教学大纲-要点摘要.md`）。
  2. **与 GOC 的映射思路**：GOC 支持 **图形输出（pen 画图）** 与 **cin/cout 输入输出**，因此不限于「画图」题——可关联更多数学知识点：
     - **画图题**：轴对称、正多边形、圆、坐标、规律与数列（用轨迹呈现）等；
     - **输入输出题**：四则运算、比与比例、方程与代数式、数列与规律（读入 n 输出结果或按结果画图）、统计（读入数据输出统计量或画统计图）等。
     便于后续按 06/07 范式出题并打 curriculum/tags。

---

## 提取建议

1. **结构优先**：先摘出目录级结构（学段 → 年级 → 模块/章 → 节/主题），再填核心要求。
2. **可操作**：每条要点尽量可转化为「一道或一类 GOC 题」的描述。题型可包括：**纯画图**（如轴对称图案、等分圆周）、**读入→计算→输出**（如 cin 读 n，cout 输出 n 的倍数或简单算式结果）、**读入→计算→按结果画图**（如 cin 读 n，画 n 边形或 n 个圆点）等。
3. **与现有标签衔接**：当前 OJ 已有数学章节（正多边形、轴对称、圆与弧、坐标系、螺旋线、色彩规律、旋转等），摘要中可标注与这些标签的对应关系，便于新题归类。

---

## 注意事项

- 若 PDF 为扫描件或图像页，正文可能无法直接复制，需 OCR 或人工摘录；本仓库中的 PDF 若为图像版，可先做目录与关键页摘录。
- 提取完成后，出题时引用「大纲要点摘要」中的模块与年级，题目 JSON 的 `tags` / `curriculum` 与 08、11 中现有数学章节体系保持一致。
