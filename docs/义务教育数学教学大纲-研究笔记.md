### 2.2 常见领域划分（课标通常采用）

义务教育数学课标通常按 **四大领域** 组织内容：

1. **数与代数**：数感、运算、式与方程、正反比例、函数等。  
2. **图形与几何**：图形的认识、测量、图形的运动与位置、证明等。  
3. **统计与概率**：数据的收集与整理、统计图、概率初步。  
4. **综合与实践**：跨领域主题、项目式学习。

### 2.3 与「画图」强相关的知识点（图形与几何 + 部分代数/规律）

从 **GOC 以「画图」为主** 的角度，下列知识点较易转化为 OJ 题目：

## 2.4 格局打开：cin/cout 带来的扩展 — 可关联更多数学知识点

GOC 支持 **cin / cout**（GOC-008 已落地），题目形态不限于「画图」：

- **纯输出（cout）**：读入数据 → 计算 → 用 cout 输出结果（算式、个数、简单统计量等），可覆盖**数与代数**中大量「运算、式、简单应用」。
- **读入→按结果画图**：已有题型（如输入 n 画 n 边形、输入 n 和颜色画 n 色花环），把**代数/几何**结合（参数化图形）。
- **读入→计算→输出 + 可选画图**：例如读入多组数据输出表格或统计结果，或先算再画（如比例分配后画条形图）。

在此基础上，**可关联的数学知识点**明显多于「仅画图」：

| 领域         | 典型主题             | 学段       | 与 GOC 的衔接（含 cin/cout） |
|--------------|----------------------|------------|------------------------------|
| **数与代数** | 四则运算、估算       | 1–2 起     | cin 读入数，cout 输出算式结果或比较结果 |
| **数与代数** | 倍数、因数、整除     | 3–4、5–6   | 读入 n，输出 n 的因数/倍数个数或列表（cout） |
| **数与代数** | 分数、小数运算       | 3–4、5–6   | 读入分子分母或小数，输出运算结果 |
| **数与代数** | 比与比例、正反比例   | 5–6、初中  | 读入两组量，输出比值或判断是否成比例；或按比例画图（如 n 份中取 k 份画 k 个圆） |
| **数与代数** | 式与方程             | 初中       | 读入系数，输出方程解或代数式化简结果（有限范围内） |
| **数与代数** | 数列与规律           | 全学段     | 读入 n，输出第 n 项或前 n 项和；或读入 n 画「第 n 个图形」（数形结合） |
| **统计与概率** | 数据的收集与整理   | 5–6、初中  | 读入一组数据，cout 输出均值/最大最小/个数，或画柱状图/折线图（pen） |
| **统计与概率** | 统计图               | 5–6、初中  | 读入数据 → 计算 → 用坐标+矩形/线段画统计图（已有思路，可加强） |
| **综合与实践** | 简单建模、方案选择 | 各学段     | 读入条件，输出「最少/最多」或一种方案（cout），或画出方案示意图（pen） |

**结论**：把「画图 + 输入输出」都算上，GOC 可关联的数学知识点可覆盖**四大领域**中的多数「可计算、可表达」内容，而不只限于图形与几何。出题与 curriculum 映射时，可对**数与代数、统计与概率**单独建章节或 tags（如「运算与式」「比与比例」「统计与数据」），与现有「正多边形、轴对称、圆与弧、坐标系、螺旋线、色彩规律、旋转」并列，便于老师按大纲选课。

1. **未从 PDF 原文摘录**：因 PDF 为图像版，本研究未对 189 页内容做逐字提取或 OCR，所有「典型结构」「典型主题」均基于**公开课标/大纲的通用描述**及**提取说明中的目标**推导。
