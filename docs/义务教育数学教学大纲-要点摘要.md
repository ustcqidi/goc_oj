# 义务教育数学教学大纲 — 要点摘要（GOC-034）

> **说明**：`docs/义务教育数学教学大纲.pdf` 为图像版扫描件，无法直接提取文字。本文档依据《义务教育数学课程标准（2022年版）》通用结构整理，部分描述参考课标公开内容，标注「待补充」处建议结合实际 PDF 补全。

> **重要补充（2026-03-18 修订）**：GOC 同时支持 `cin`（输入）和 `cout`（文字/数字输出），不仅限于图形绘制。这意味着 GOC 可以出**计算类题目**（输入数据 → 输出计算结果），覆盖大纲中「数与代数」「统计与概率」等原本无法用画图表达的知识点。本文档已将两种题型（图形输出、计算输出）同步纳入衔接分析。

---

## 一、总体框架

| 学段 | 年级 | 核心目标 |
|------|------|---------|
| 第一学段 | 1–2 年级 | 数感、形状感知、简单规律 |
| 第二学段 | 3–4 年级 | 四则运算、图形认识、统计初步 |
| 第三学段 | 5–6 年级 | 分数/小数运算、面积体积、比例、坐标入门 |
| 第四学段 | 7–9 年级（初中） | 代数、几何（相似、圆、三角函数）、统计与概率 |

---

## 二、GOC 两种输出模式

| 模式 | 语法 | 判题方式 | 适合知识点 |
|------|------|---------|-----------|
| **图形输出** | `pen.fd/rt/o/oo/...` | 轨迹匹配（expectedTrajectory） | 几何图形、对称、旋转、坐标系 |
| **计算输出** | `cout << 结果` | 标准输出匹配（expectedOutput） | 四则运算、面积公式、数列、统计、勾股定理 |

两种模式可组合：`cin` 读入参数 → 计算 → `cout` 输出结果，或 `cin` 读入参数 → 画图。

## 三、各学段核心知识点与 GOC 衔接

### 第一学段（1–2 年级）

#### 数与代数
- 20 以内加减法、100 以内加减法
- 认识整数，简单乘法口诀

#### 图形与几何
- **认识平面图形**：三角形、正方形、长方形、圆（基本形状）
- **对称初步**：图形的对折（轴对称初步感知）
- 方向与位置：上下左右前后

#### 规律与统计
- 发现简单数列规律（如 2、4、6、8…）

**→ GOC 衔接**：
- 图形输出：画正方形、长方形、圆（`001-008`、正多边形系列）
- 计算输出：`cin >> a >> b; cout << a + b;`（加减法验算）；输出乘法口诀表（嵌套循环 + `cout`）

---

### 第二学段（3–4 年级）

#### 数与代数
- 三位数加减，乘除法（包括两位数乘两位数）
- 分数初步认识

#### 图形与几何
- **角的认识**：锐角、直角、钝角、平角
- **三角形与四边形**：等腰三角形、等边三角形、平行四边形
- **轴对称图形**：认识轴对称，画对称轴
- **周长与面积**：长方形/正方形的周长与面积公式
- 平移与旋转初步

**→ GOC 衔接**：
- 图形输出：等边三角形 → `001`；轴对称图案 → `tag:轴对称`；旋转 → `tag:旋转`
- 计算输出：`cin >> a >> b; cout << 2*(a+b);`（输入长宽求周长）；输入边长求面积（`cout << a*b`）

---

### 第三学段（5–6 年级）

#### 数与代数
- 分数/小数四则运算、比与比例
- **正比例与反比例**（变化规律）

#### 图形与几何
- **多边形面积**：三角形、梯形、多边形
- **圆的认识**：直径、半径、圆周率、圆的周长与面积
- **等分圆**：等分圆周（为正多边形埋下伏笔）
- **立体图形初步**：长方体、正方体、圆柱、圆锥（表面积与体积）
- **坐标系初步**：建立平面直角坐标系，描述图形位置

**→ GOC 衔接**：
- 图形输出：圆 → `005、016、021、023、036、037`；等分圆周 → `016、044`；坐标系 → `tag:坐标系`
- 计算输出：输入底和高求三角形面积；输入半径求圆周长/面积（`cout << 2*3.14159*r`）；输入 a/b/h 求梯形面积；输入长宽高求长方体体积

---

### 第四学段（7–9 年级，初中）

#### 数与代数
- 有理数、实数、代数式化简
- **方程与不等式**：一元一次方程、二元一次方程组、二次方程
- **函数**：一次函数、二次函数、反比例函数，图像与特征
- **数列规律**（等差数列初步）

#### 图形与几何
- **全等三角形**（SSS、SAS、ASA、AAS）
- **相似三角形**与比例线段
- **勾股定理**
- **圆的性质**：圆心角、弦、切线、弧长、扇形面积
- **正多边形与圆**：正多边形的外接圆与内切圆
- **三角函数**（初步）：sin、cos、tan 的基本概念
- **空间图形**：棱柱、棱锥、圆柱、圆锥的表面积与体积

#### 统计与概率
- 数据收集、整理（频率直方图、折线图）
- 概率初步（古典概型）

**→ GOC 衔接**：
- 图形输出：正多边形 → `001-003、008、032`；圆与弧 → `005、016、021、023`；坐标折线图 → `tag:坐标系`；螺旋（等差数列）→ `tag:螺旋`
- 计算输出：输入 a/b 求斜边（`sqrt(a*a+b*b)`）；输出等差数列前 n 项；输入 n 组数据求平均值/最大值；输出 Fibonacci 数列；判断质数；求最大公约数（辗转相除）

## 七、Schema 扩展建议

为支持计算输出类题目，建议在题目 JSON 中增加 `expectedOutput` 字段：

```json
{
  "id": "151",
  "title": "勾股定理：求斜边",
  "description": "输入直角三角形两直角边 a、b，输出斜边长（保留2位小数）。",
  "difficulty": "easy",
  "tags": ["cin输入", "计算输出", "勾股定理"],
  "starterCode": "int a, b;\ncin >> a >> b;\n// 计算并输出斜边\n",
  "solution": "int a, b;\ncin >> a >> b;\ncout << fixed;\ncout.precision(2);\ncout << sqrt((double)a*a + (double)b*b);",
  "defaultInput": "3 4",
  "expectedOutput": "5.00",
  "expectedTrajectory": []
}
```

判题逻辑：若 `expectedOutput` 非空，则比对 `cout` 输出字符串（trim 后匹配）；`expectedTrajectory` 非空则比对轨迹；两者可并存。
