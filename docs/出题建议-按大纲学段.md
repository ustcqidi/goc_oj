**描述：** 输入 n 和 n 个数据值（1≤每值≤10），画 n 根柱子（柱宽 20，数据值乘以 10 作为像素高度），颜色循环，底边对齐 y=-60，从 x=-n×15 开始排列。

```cpp
int n, v;
cin >> n;
pen.c(0).size(1).moveTo(-120,-60).lineTo(120,-60);
for(int i = 0; i < n; i++){
    cin >> v;
    int x = -n*15 + i*30;
    pen.c(i%6+1);
    pen.moveTo(x,-60).lineTo(x,-60+v*10).lineTo(x+20,-60+v*10).lineTo(x+20,-60);
}
```

### P-060 水仙花数（自幂数）
| 项 | 内容 |
|----|------|
| 年级 | 7–9 年级 |
| 知识点 | 幂运算、数字分解 |
| 输出模式 | 算 |
| 难度 | hard |
| tags | `计算输出` `循环` `取余` `幂运算` |

**描述：** 无输入，输出所有三位水仙花数（各位数字的立方和等于该数本身，如 153 = 1³+5³+3³），空格分隔。

```cpp
int i, a, b, c;
int first = 1;
for(i = 100; i <= 999; i++){
    a = i / 100;
    b = (i / 10) % 10;
    c = i % 10;
    if(a*a*a + b*b*b + c*c*c == i){
        if(!first) cout << " ";
        cout << i;
        first = 0;
    }
}
```
