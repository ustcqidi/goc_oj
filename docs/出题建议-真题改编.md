# GOC 出题建议 — 真题改编版

> 基于义务教育各学段数学考试真题（期末考、中考、小学奥赛），改编为 GOC 编程题。
>
> **格式说明：** 每道题包含原题来源、核心知识点、GOC 改编后的输入/输出、参考解法（GOC 简化 C++）。
>
> **GOC 语法约束：** `int`/`double` 变量、`for`/`while`/`if-else`、`cin`/`cout`、`pen.*`，无自定义函数。

### T-01 加减法口算（期末真题）
**原题来源：** 小学一年级数学期末常见题型
**原题：** 计算 37 + 48 = ？，82 - 35 = ？

**GOC 改编：** 输入两个正整数 a 和 b，输出它们的和与差（两行输出）。

| 项 | 值 |
|----|-----|
| 年级 | 1–2 年级 |
| 知识点 | 100 以内加减法 |
| 难度 | easy |
| tags | `cin输入` `计算输出` `加法` `减法` |

**输入示例：** `37 48`
**输出：**
```
85
-11
```

```cpp
int a, b;

```

### T-04 数字规律：找第 n 个奇数（小学奥赛）
**原题来源：** 小学奥数经典题
**原题：** 1, 3, 5, 7, 9, … 第 n 个奇数是多少？前 n 个奇数的和是多少？

**GOC 改编：** 输入 n，输出第 n 个奇数和前 n 个奇数之和（空格分隔）。规律：第 n 个奇数 = 2n-1；前 n 个奇数之和 = n²。

| 项 | 值 |
|----|-----|
| 年级 | 1–2 年级 |
| 知识点 | 奇数规律、数列 |
| 难度 | easy |
| tags | `cin输入` `计算输出` `数列` `规律` |

**输入示例：** `5`
**输出：** `9 25`

```cpp
int n, sum;
cin >> n;
sum = 0;
for(int i = 1; i <= n; i++) sum = sum + (2*i - 1);
cout << 2*n - 1 << " " << sum;
```

### T-05 画彩色规律图案（一年级图形认识）
**原题来源：** 小学一年级数学教材中的"认识图形"
**原题：** 用三角形、圆形、正方形排列规律图案。

**GOC 改编：** 画一排 6 个图形：圆、三角、正方形循环两遍，颜色依次为红、绿、蓝。

| 项 | 值 |
|----|-----|
| 年级 | 1–2 年级 |
| 知识点 | 认识平面图形、规律 |
| 难度 | easy |
| tags | `图形` `循环` `颜色` `规律` |

```cpp
for(int i = 0; i < 6; i++){
    int x = -125 + i * 50;
    pen.c(i%3 + 1);
    if(i%3 == 0) pen.moveTo(x, 0).oo(15);
    if(i%3 == 1){
        pen.moveTo(x, -15).lineTo(x-15, 15).lineTo(x+15, 15).lineTo(x, -15);
    }
    if(i%3 == 2){
        pen.moveTo(x-15, -15).lineTo(x+15, -15).lineTo(x+15, 15).lineTo(x-15, 15).lineTo(x-15, -15);
    }
}
```

### T-14 鸡兔同笼（小学奥赛必考经典）
**原题来源：** 《孙子算经》经典，小学四年级奥数必考
**原题：** 笼中有鸡和兔共 35 头，共 94 只脚，问鸡兔各几只？

**GOC 改编：** 输入总头数 h 和总脚数 f，用枚举法求鸡数和兔数，输出"鸡数 兔数"，无解输出 `no`。

| 项 | 值 |
|----|-----|
| 年级 | 3–4 年级 |
| 知识点 | 鸡兔同笼、枚举、方程思想 |
| 难度 | medium |
| tags | `cin输入` `计算输出` `循环` `枚举` `鸡兔同笼` |

**输入示例：** `35 94`
**输出：** `23 12`

```cpp
int h, f, found;
cin >> h >> f;
found = 0;
for(int c = 0; c <= h; c++){
    int r = h - c;
    if(2*c + 4*r == f){
        cout << c << " " << r;
        found = 1;
    }
}
if(!found) cout << "no";
```

### T-43 奇偶图形规律（一年级图形规律）
**原题：** △ □ △ □ △ ___，下一个是什么？

**GOC 改编：** 画一排 8 个图形，奇数位画红色三角形，偶数位画蓝色正方形，用 `if(i%2==0)` 切换。

| 项 | 值 |
|----|-----|
| 年级 | 1–2 年级 | 难度 | easy |
| tags | `循环` `条件判断` `正多边形` `颜色` `规律` |

```cpp
for(int i = 0; i < 8; i++){
    int x = -105 + i*30;
    pen.up().moveTo(x, 0).down();
    if(i%2 == 0){
        pen.c(1);
        pen.fd(20).rt(120).fd(20).rt(120).fd(20).rt(120);
    } else {
        pen.c(6);
        for(int j = 0; j < 4; j++) pen.fd(20).rt(90);

}
```

### T-45 高斯求和点阵（七年级等差求和可视化）
