# 002 - 螺旋正方形

## 题目描述

画下面正方形螺旋线，长度 0、10、20、30…190

## 绘图要求

- 颜色：蓝色（`pen.c(2)`）
- 线条粗细：默认（1）
- 转角：每次右转 90°
- 边长规律：第 i 条边长度为 `i * 10`（i 从 0 到 19，共 20 条）
- 必须使用 `for` 循环完成

## 期望输出

见 `output.png`

## 难度

入门

## 标签

`循环` `正多边形` `角度` `等差数列`

## 参考答案

```cpp
pen.c(2);
for(int i = 0; i < 20; i++){
    pen.fd(i * 10).rt(90);
}
```
