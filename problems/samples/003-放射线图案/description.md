# 003 - 放射线图案

## 题目描述

从同一个起点出发，画出如图所示的放射线图案。

## 绘图要求

- 颜色：橙色（`pen.c(14)`）
- 线条粗细：默认（1）
- 共 12 条射线，相邻两条间隔 30°（12 × 30° = 360°，恰好转满一圈）
- 每条射线长度依次为 15、30、45…180（等差，公差 15）
- 每条射线画完后退回起点，再转向画下一条
- 必须使用 `for` 循环完成

## 期望输出

见 `output.png`

## 难度

入门

## 标签

`循环` `角度` `等差数列` `bk退回`

## 参考答案

```cpp
pen.c(14);
for(int i = 0; i < 12; i++){
    pen.fd((i + 1) * 15).bk((i + 1) * 15).rt(30);
}
```
