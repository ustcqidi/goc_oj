# 008 - 画正方形（下册版）

**来源**：https://noi.hnai.net/problem/269.html（椰程信奥，下册第1课 P269）

## 题目描述

边长200的正方形。使用 for 循环，循环4次，每次前进200步，右转90°，画出一个边长为200的正方形。

## 要求

- 使用 for 循环画正方形
- 边长为 200
- 转角为 90°，循环 4 次

## 期望输出

见 `output.png`（一个边长200的正方形，从中心开始绘制）

## 难度

入门

## 标签

`循环` `正多边形` `角度`

## 参考答案

```cpp
int main()
{
    for(int i = 0;i < 4 ; i ++){
        pen.fd(200).rt(90);
    }
    return 0;
}
```

> ✅ 此题使用标准 GOC 绘图命令，与 OJ 执行器兼容。
