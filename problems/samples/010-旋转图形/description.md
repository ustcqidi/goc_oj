# 010 - 旋转图形

**来源**：https://noi.hnai.net/problem/277.html（椰程信奥，下册第2课 P277）

## 题目描述

输入n的值，画一个由n个基本图形旋转而成的图形。当n小于5时显示小鸟，等于5时显示猴子，大于5时显示蝴蝶。线段长度100。

核心几何逻辑：循环n次，每次前进100步（显示图片）、后退100步、右转 360/n°。形成n折旋转对称图案。

## 要求

- 读入整数 n
- 循环 n 次，每次 `fd(100).bk(100).rt(360.0/n)`
- if 条件选择不同图片

## 期望输出

见 `output.png`（n折旋转对称图案，中心发散）

## 难度

进阶

## 参考答案

```cpp
int main(){
    p.picL( 1,"bird.png");
    p.picL( 2,"betterfly.png");
    p.picL( 3,"monkey.png");
    int n , a;
    cin >> n;
    if(n<5) a = 1;
    if(n == 5) a = 3;
    if(n > 5) a = 2;

    for(int i = 0; i<n;i++){
        p.fd(100).pic(a).bk(100);
        pen.rt(360.0/n);
    }
    return 0;
}
```

标准GOC等价版（无图片，用 o 标记）：
```cpp
int n;
cin >> n;
for(int i = 0; i < n; i++){
    pen.fd(100).o(8).bk(100).rt(360.0/n);
}
```

> ⚠️ 原题使用扩展命令 + cin，OJ 不支持。标准等价版可在 OJ 中运行（如不含 cin 则改为固定值）。
