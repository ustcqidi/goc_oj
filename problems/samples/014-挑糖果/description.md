# 014 - 挑糖果

**来源**：https://noi.hnai.net/problem/283.html（椰程信奥，下册第3课 P283）

## 题目描述

小C分到10颗糖果，每颗有类型（1=软糖，2=硬糖）和颜色（0-14）。循环读入10行，如果是软糖（a==1）则用对应颜色画一个半径30的实心圆，然后前进60步（排成一排）。

## 要求

- 循环10次，每次读入 a（类型）和 col（颜色）
- if(a==1) 则 pen.oo(30, col).fd(60)（画软糖）
- 画笔右转90°并抬起作为初始朝向

## 期望输出

见 `output.png`（一排彩色实心圆代表软糖）

## 难度

进阶

## 标签

`循环` `条件` `输入输出` `颜色` `圆形`

## 参考答案

```cpp
int main()
{
   int a, col;
   pen.rt(90).up().hide();
   for(int i=0; i<10; i++)
   {
       cin >> a >> col;
       if(a == 1)
           pen.oo(30, col).fd(60);
   }
}
```

> ⚠️ 原题使用 `cinWin()` 扩展函数和 `oo(30, col)` 双参数形式（OJ 执行器不支持 col 参数），含 cin，仅作参考样例。
