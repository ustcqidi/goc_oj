# 017 - 挂灯笼

**来源**：https://noi.hnai.net/problem/287.html（椰程信奥，下册第4课 P287）

## 题目描述

画出7个灯笼排成一排。循环7次，每次用 `i%3` 决定显示哪种灯笼图片（0、1、2三种循环），然后前进100步。

核心模式：`i%3` 实现3种内容周期轮换，配合循环和前进构成一排渐变图案。

## 要求

- 起始：右转90°、抬笔、后退300步
- 循环7次，i%3 决定图片编号（0/1/2循环）
- 每次显示图片后前进100步

## 期望输出

见 `output.png`（7个灯笼横向排列，3种样式循环）

## 难度

进阶

## 标签

`循环` `取余` `倍数关系` `扩展命令`

## 参考答案

```cpp
int main()
{
   pen.up().rt(90).bk(300);
   for(int i=0; i<7; i++){
      // pen.pic(i % 3);  // 原题使用图片
      pen.c(i % 3 + 1).oo(20).fd(60);  // 标准等价：3种颜色循环
   }
   return 0;
}
```

标准GOC等价：
```cpp
pen.up().rt(90).bk(300);
for(int i=0; i<7; i++){
    pen.c(i % 3 + 1).oo(20).fd(60);
}
```

> ⚠️ 原题使用 picL/pic 显示灯笼图片，OJ 不支持。标准等价版用颜色循环替代图片轮换。
