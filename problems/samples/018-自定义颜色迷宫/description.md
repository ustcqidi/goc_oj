# 018 - 自定义颜色的迷宫

**来源**：https://noi.hnai.net/problem/286.html（椰程信奥，下册第4课 P286）

## 题目描述

画一个边长逐渐变长的15边回旋图（螺旋方形迷宫），边长依次为10、20、30……150，颜色紫色（11号）、绿色（3号）周期交替变化。

## 要求

- 变量 len 从 10 开始
- 循环15次：奇偶判断换色（i%2==0用11号，i%2==1用3号），前进len，右转90°，len加10
- 形成方形螺旋迷宫图案

## 期望输出

见 `output.png`（双色交替的方形螺旋迷宫）

## 难度

进阶

## 标签

`循环` `条件` `等差数列` `变量计算` `颜色` `螺旋图案`

## 参考答案

```cpp
int main()
{
    int len = 10;
    for(int i = 0; i < 15; i++)
    {
        if(i % 2 == 0) pen.c(11);
        if(i % 2 == 1) pen.c(3);
        pen.fd(len).rt(90);
        len = len + 10;
    }
    return 0;
}
```
