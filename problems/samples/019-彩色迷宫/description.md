# 019 - 彩色迷宫

**来源**：https://noi.hnai.net/problem/285.html（椰程信奥，下册第4课 P285）

## 题目描述

画一个边长逐渐变长的15边回旋图，边长依次为10、20、30……，颜色黑（0号）、红（1号）周期变化（`i%2`）。

## 要求

- 变量 len 从 10 开始
- 循环15次：`pen.c(i%2)` 设颜色，前进len，右转90°，len加10
- 比自定义颜色迷宫更简洁：直接用 `c(i%2)` 无需 if

## 期望输出

见 `output.png`（黑红交替的方形螺旋迷宫）

## 难度

进阶

## 标签

`循环` `取余` `等差数列` `颜色` `螺旋图案` `变量计算`

## 参考答案

```cpp
int main()
{
    int len = 10;
    for(int i = 0; i < 15; i++)
    {
        pen.c(i % 2);
        pen.fd(len).rt(90);
        len = len + 10;
    }
    return 0;
}
```
