# 020 - 画迷宫

**来源**：https://noi.hnai.net/problem/284.html（椰程信奥，下册第4课 P284）

## 题目描述

画一个边长逐渐变长的15边回旋图，边长依次为10、20、30……，单色（无颜色切换）。是螺旋四边形的最简洁版本。

## 要求

- 变量 len 从 10 开始
- 循环15次：前进len，右转90°，len加10

## 期望输出

见 `output.png`（单色方形螺旋迷宫）

## 难度

入门

## 标签

`循环` `等差数列` `变量计算` `螺旋图案`

## 参考答案

```cpp
int main()
{
    int len = 10;
    for(int i = 0; i < 15; i++)
    {
        pen.fd(len).rt(90);
        len = len + 10;
    }
    return 0;
}
```
