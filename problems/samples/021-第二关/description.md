# 021 - 第二关（抽取10的倍数）

**来源**：https://noi.hnai.net/problem/293.html（椰程信奥，下册第5课 P293）

## 题目描述

从1到50中抽取所有10的倍数号码（10,20,30,40,50）横向展示。循环步长为10：`for(int i=10; i<=50; i=i+10)`。

## 要求

- for 循环：起始 i=10，步长 10，终止 i<=50
- 每次显示号码并前进150步

## 期望输出

见 `output.png`（5个号码横向排列：10,20,30,40,50）

## 难度

进阶

## 标签

`循环` `步长` `倍数关系` `输入输出`

## 参考答案

```cpp
// 标准GOC等价（用oo画圆代替号码显示）
pen.up().rt(90).bk(300);
for(int i = 10; i <= 50; i = i + 10){
    pen.c(i/10).oo(20).fd(100);
}
```

> ⚠️ 原题使用 picL/pic/text 扩展命令，OJ 不支持。标准等价版可在 OJ 运行。
