# 022 - 第一关（抽取奇数）

**来源**：https://noi.hnai.net/problem/292.html（椰程信奥，下册第5课 P292）

## 题目描述

从1到10中抽取所有奇数号码（1,3,5,7,9）横向展示。循环步长为2：`for(int i=1; i<10; i=i+2)`。

## 要求

- for 循环：起始 i=1，步长 2，终止 i<10
- 每次显示号码并前进150步

## 期望输出

见 `output.png`（5个奇数号码横向排列：1,3,5,7,9）

## 难度

进阶

## 标签

`循环` `步长` `奇数` `输入输出`

## 参考答案

```cpp
// 标准GOC等价（用oo画圆代替号码显示）
pen.up().rt(90).bk(300);
for(int i = 1; i < 10; i = i + 2){
    pen.c(i).oo(20).fd(100);
}
```

> ⚠️ 原题使用 picL/pic/text 扩展命令，OJ 不支持。标准等价版可在 OJ 运行。
