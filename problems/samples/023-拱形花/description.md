# 023 - 拱形花

**来源**：https://noi.hnai.net/problem/291.html（椰程信奥，下册第5课 P291）

## 题目描述

画一组从远到近逐步增大的拱形花，第一个宽和高都是100，后面每个依次增加200，直到宽高小于800。即循环步长为200：`for(int i=100; i<800; i=i+200)`，共4次（100,300,500,700）。

## 要求

- for 循环：起始 i=100，步长 200，终止 i<800
- 每次显示大小为 (i,i) 的图案（渐大）

## 期望输出

见 `output.png`（4个渐大的同心拱形花）

## 难度

进阶

## 标签

`循环` `步长` `等差数列` `同心图形` `扩展命令`

## 参考答案

```cpp
// 标准GOC等价（用o画空心圆代替图片，半径从50递增）
for(int i = 100; i < 800; i = i + 200){
    pen.o(i/2);
}
```

> ⚠️ 原题使用 picL/pic 扩展命令。标准等价版用同心圆表示渐大效果。
