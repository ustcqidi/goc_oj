# 026 - 螺旋三角形2（小角度平滑螺旋）

**来源**：https://noi.hnai.net/problem/288.html（椰程信奥，下册第5课 P288）

## 题目描述

画一条平滑渐大的螺旋三角形线，边长从0开始，每步加10（`fd(i*10)`），每步右转2°（极小转角）。共200步。因为每步转角仅2°，图形看起来像平滑曲线而非折线。

## 要求

- 颜色1（红色）
- for 循环200次（i=0到199），步长1
- 每次 `pen.fd(i*10).rt(2)`

## 期望输出

见 `output.png`（红色平滑螺旋线）

## 难度

进阶

## 标签

`循环` `螺旋图案` `等差数列` `变量计算`

## 参考答案

```cpp
int main()
{
   pen.c(1);
   for(int i = 0; i < 200; i = i + 1){
       pen.fd(i * 10).rt(2);
   }
   return 0;
}
```

> ✅ 纯标准 GOC 命令，完全兼容 OJ 执行器！注意：200次迭代可能较慢，建议设置极速模式运行。
