# 027 - 画十字（无循环版）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P28）

## 题目描述

画一个十字形（加号）图案。从中心出发，向上画50步后退回，右转90°，重复4次，形成上下左右各一条臂的十字。无循环，直接4次重复语句。

## 要求

- 4次 `p.fd(50).bk(50).rt(90)`
- 形成十字形（每臂长50）

## 期望输出

见 `output.png`（十字形）

## 难度

入门

## 标签

`前进后退` `抬笔落笔` `对称`

## 参考答案

```cpp
int main() {
    pen.fd(50).bk(50).rt(90);
    pen.fd(50).bk(50).rt(90);
    pen.fd(50).bk(50).rt(90);
    pen.fd(50).bk(50).rt(90);
    return 0;
}
```

循环等价写法：
```cpp
for(int i = 0; i < 4; i++){
    pen.fd(50).bk(50).rt(90);
}
```

> ✅ 纯标准 GOC 命令，完全兼容 OJ 执行器！
