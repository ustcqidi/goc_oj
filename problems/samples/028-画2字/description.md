# 028 - 画2字（倾斜正方形/菱形）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P27）

## 题目描述

先右转45°（倾斜方向），再用循环画一个边长50的正方形，结果是一个菱形（倾斜45°的正方形）。

## 要求

- `p.rt(45)` 初始右转45°
- for 循环4次：`p.fd(50).rt(90)`
- 形成菱形（45°倾斜的正方形）

## 期望输出

见 `output.png`（菱形）

## 难度

入门

## 标签

`循环` `正多边形` `角度` `初始方向`

## 参考答案

```cpp
int main() {
    pen.rt(45);
    for(int i = 0; i < 4; i++)
        pen.fd(50).rt(90);
    return 0;
}
```
