# 029 - 画旗帜

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P29）

## 题目描述

先直线前进100步（旗杆），再用循环画边长50的正方形（旗面）。结合直线运动和循环多边形。

## 要求

- `p.fd(100)` 画旗杆
- `for(int i=0;i<4;i++) p.fd(50).rt(90)` 画旗面

## 期望输出

见 `output.png`（旗杆 + 正方形旗面）

## 难度

入门

## 标签

`循环` `正多边形` `组合图形`

## 参考答案

```cpp
int main() {
    pen.fd(100);
    for(int i = 0; i < 4; i++)
        pen.fd(50).rt(90);
    return 0;
}
```
