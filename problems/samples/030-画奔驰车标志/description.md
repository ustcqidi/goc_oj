# 030 - 画奔驰车标志（三叉星）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P30）

## 题目描述

画奔驰车标志——三叉星：从中心出发，向三个方向各伸出一条臂（间距120°）。3次 `p.fd(50).bk(50).rt(120)`。

## 要求

- 3次：`pen.fd(50).bk(50).rt(120)`
- 形成3折对称的三叉星标志

## 期望输出

见 `output.png`（三叉星，3条臂120°均匀分布）

## 难度

入门

## 标签

`前进后退` `旋转对称` `对称`

## 参考答案

```cpp
int main()
{
   pen.fd(50).bk(50).rt(120);
   pen.fd(50).bk(50).rt(120);
   pen.fd(50).bk(50).rt(120);
}
```

循环等价写法：
```cpp
for(int i = 0; i < 3; i++){
    pen.fd(50).bk(50).rt(120);
}
```
