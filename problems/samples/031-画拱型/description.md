# 031 - 画拱型（U形）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P31）

## 题目描述

用三段直线画一个拱型（U形/门拱）：前进100步，右转90°，前进50步，右转90°，前进100步。无循环，纯顺序语句。

## 要求

- `pen.fd(100).rt(90).fd(50).rt(90).fd(100)`
- 形成三边的U形拱

## 期望输出

见 `output.png`（U形拱，左竖100，底横50，右竖100）

## 难度

入门

## 标签

`前进后退` `转向` `组合图形`

## 参考答案

```cpp
int main() {
    pen.fd(100).rt(90).fd(50).rt(90).fd(100);
    return 0;
}
```
