# 032 - 画正三角形（初始方向调整版）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P32）

## 题目描述

先右转30°调整方向，再用循环画等边三角形（3次 fd(100).rt(120)）。初始转角影响三角形朝向。

## 要求

- `pen.rt(30)` 调整初始方向
- `for(int i=0;i<3;i++) pen.fd(100).rt(120)`
- 画出朝向调整后的等边三角形

## 期望输出

见 `output.png`（初始右转30°的等边三角形）

## 难度

入门

## 标签

`循环` `正多边形` `角度` `初始方向`

## 参考答案

```cpp
int main() {
    pen.rt(30);
    for(int i = 0; i < 3; i++)
        pen.fd(100).rt(120);
    return 0;
}
```
