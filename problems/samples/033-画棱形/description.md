# 033 - 画棱形（倾斜正方形）

**来源**：https://noi.hnai.net/problem/307.html（椰程信奥，魔法学院第3课 P33）

## 题目描述

与028（画2字）相同：先右转45°，再用循环画正方形，得到菱形。两道题考察相同概念，对比可加深理解。

## 要求

- `pen.rt(45)` 初始右转
- `for(int i=0;i<4;i++) pen.fd(50).rt(90)` 画正方形
- 结果是45°倾斜的正方形（菱形）

## 期望输出

见 `output.png`（菱形）

## 难度

入门

## 标签

`循环` `正多边形` `角度` `初始方向`

## 参考答案

```cpp
int main() {
    pen.rt(45);
    for(int i = 0; i < 4; i++)
        pen.fd(50).rt(90);
    return 0;
}
```
