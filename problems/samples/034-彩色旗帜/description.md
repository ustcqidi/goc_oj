# 034 - 画彩色旗帜（魔法学院第4课）

**来源**：https://noi.hnai.net/problem/308.html（椰程信奥，魔法学院第4课 P308）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形。

画一面彩色旗帜：旗杆棕色(颜色6)，长80步；旗面由三条水平彩色色带构成，自下而上颜色分别为1(红)、2(绿)、3(蓝)，每色带宽20步、长60步。

## 要求

- `pen.c(6).fd(80)` 画棕色旗杆（朝上）
- 右转90°后，用三段长60步的线段画三色旗面
- 每段颜色不同（颜色1、2、3）

> ⚠️ 原页面代码块为空（未提供），以下为推断重构版本。

## 期望输出

## 参考答案

```cpp
int main() {
    pen.c(6).fd(80);         // 旗杆（棕色）
    pen.rt(90);
    pen.c(1).fd(60).bk(60); // 旗面1（红）
    pen.lt(90).fd(20).rt(90);
    pen.c(2).fd(60).bk(60); // 旗面2（绿）
    pen.lt(90).fd(20).rt(90);
    pen.c(3).fd(60).bk(60); // 旗面3（蓝）
    return 0;
}
```
