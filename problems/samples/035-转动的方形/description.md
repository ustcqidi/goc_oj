# 035 - 画转动的方形（魔法学院第4课）

**来源**：https://noi.hnai.net/problem/308.html（椰程信奥，魔法学院第4课 P308）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形。

从里到外画7段递增的折线，形成"转动方形"效果：
- 线段长度依次为：20, 30, 40, 50, 60, 70, 80
- 颜色号依次为：0, 1, 2, 3, 4, 5, 6
- 线的宽度(size)依次为：2, 3, 4, 5, 6, 7, 8
- 每段之间右转 90°

## 要求

- 循环7次，每次右转90°再前进 `20+i*10` 步
- 同步设置颜色 `i` 和宽度 `i+2`

## 期望输出

见原题图（旋转螺旋折线效果）

## 难度

入门

## 标签

`循环` `颜色` `size` `螺旋` `变步长`

## 参考答案

```cpp
int main() {
    int a = 20;
    for (int i = 0; i < 7; i++) {
        pen.c(i).size(i + 2).rt(90).fd(a + i * 10);
    }
    return 0;
}
```
