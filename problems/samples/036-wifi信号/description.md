# 036 - 画wifi信号（魔法学院第4课）

**来源**：https://noi.hnai.net/problem/308.html（椰程信奥，魔法学院第4课 P308）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形。

从左到右画5根竖线，模拟WiFi信号格：
- 颜色号：10（同一颜色）
- 间隔距离：20步
- 线段长度依次为：20, 40, 60, 80, 100
- 线的宽度(size)依次为：2, 3, 4, 5, 6

## 要求

- 循环5次，每次先画当前高度的竖线，再移动到下一根位置
- 用 `pen.up/down` 配合 `pen.rt/lt/fd` 控制移动

## 期望输出

见原题图（从矮到高的5根竖线，类似WiFi信号格）

## 难度

入门

## 标签

`循环` `颜色` `size` `抬笔落笔` `变步长`

## 参考答案

```cpp
int main() {
    for (int i = 0; i < 5; i++) {
        pen.c(10).size(i + 2).fd((i + 1) * 20).bk((i + 1) * 20);
        if (i < 4) {
            pen.rt(90).up().fd(20).lt(90).down();
        }
    }
    return 0;
}
```
