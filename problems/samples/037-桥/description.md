# 037 - 桥（魔法学院第5课）

**来源**：https://noi.hnai.net/problem/309.html（椰程信奥，魔法学院第5课 P309）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形。

画一座桥形图案：颜色3（绿色），由4段折线组成：
- 右转90°前进30（右桥墩下段）
- 左转90°前进30（桥面左段）
- 右转90°前进60（右桥墩）
- 右转90°前进30（桥面右段）
- 左转90°前进30（左桥墩下段）

## 要求

- 颜色3，无需循环
- 用 `rt/lt` + `fd` 的折线组合绘制

## 期望输出

见原题图（类π形桥）

## 难度

入门

## 标签

`折线` `顺序执行` `颜色` `组合图形`

## 参考答案

```cpp
int main() {
    pen.c(3);
    pen.rt(90).fd(30).lt(90).fd(30);
    pen.rt(90).fd(60).rt(90).fd(30);
    pen.lt(90).fd(30);
    return 0;
}
```

> ✅ 纯标准 GOC 命令，完全兼容 OJ 执行器！
