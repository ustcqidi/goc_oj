# 038 - 转动的方形（魔法学院第5课）

**来源**：https://noi.hnai.net/problem/309.html（椰程信奥，魔法学院第5课 P309）

## 题目描述

编写程序，根据图形中红色数字指定的长度，画出下面图形。

从角落出发，每次右转90°并前进递增步数，画出螺旋方形的一角（5段折线）：
- `int a = 20`
- 线段长度依次：20, 40, 60, 80, 100（= a, a+20, a+40, a+60, a×5）

## 要求

- 初始变量 `int a = 20`
- 5次 `pen.rt(90).fd(...)` 依次递增

## 期望输出

见原题图（旋转递增折线）

## 难度

入门

## 标签

`变量` `折线` `顺序执行` `变步长`

## 参考答案

```cpp
int main() {
    int a = 20;
    pen.rt(90).fd(a);
    pen.rt(90).fd(a + 20);
    pen.rt(90).fd(a + 40);
    pen.rt(90).fd(a + 60);
    pen.rt(90).fd(a * 5);
    return 0;
}
```
