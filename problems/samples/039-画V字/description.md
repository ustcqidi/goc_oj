# 039 - 画V字（魔法学院第5课）

**来源**：https://noi.hnai.net/problem/309.html（椰程信奥，魔法学院第5课 P309）

## 题目描述

请编程，根据图形中红色数字指定的长度，画出下面图形。

画一个V字形：
- V的角度是60°（两臂夹角60°）
- 笔粗20
- `int a = 20`，臂长 = `a*10 = 200`
- 左转30°后画第一臂，回到起点，右转60°画第二臂

## 要求

- 初始变量 `int a = 20`
- `pen.lt(30)` 先左转确定V的左臂方向
- `pen.size(a)` 设置笔宽
- 前进 `a*10` 步，再后退回到顶点，右转60°，再前进 `a*10` 步

## 期望输出

见原题图（粗线V字，两臂等长，夹角60°）

## 难度

入门

## 标签

`变量` `折线` `size` `顺序执行` `角度`

## 参考答案

```cpp
int main() {
    int a = 20;
    pen.lt(30);
    pen.size(a);
    pen.fd(a * 10).bk(a * 10);
    pen.rt(60);
    pen.fd(a * 10);
    return 0;
}
```
