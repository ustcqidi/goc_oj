# 040 - 梳子（魔法学院第6课）

**来源**：https://noi.hnai.net/problem/310.html（椰程信奥，魔法学院第6课 P310）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形（梳子形）。

- 颜色6（棕色），笔宽8
- 边长 a = 40
- 循环6次，每次画一个"梳齿"：前进40，右转90，前进40，右转90，前进40，右转180（回到基线，反向）

## 要求

- `pen.size(8).c(6)`（或 `pen.size(8).color(6)`）
- `int a = 40`
- 循环6次，每次三段 fd(a).rt(90/180)

## 期望输出

见原题图（梳子，6个齿）

## 难度

入门

## 标签

`循环` `折线` `颜色` `size` `梳型`

## 参考答案

```cpp
int main() {
    int a = 40;
    pen.size(8).c(6);
    for (int i = 0; i < 6; i++) {
        pen.fd(a).rt(90);
        pen.fd(a).rt(90);
        pen.fd(a).rt(180);
    }
    return 0;
}
```
