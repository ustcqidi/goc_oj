# 041 - 锯齿（魔法学院第6课）

**来源**：https://noi.hnai.net/problem/310.html（椰程信奥，魔法学院第6课 P310）

## 题目描述

请编写程序，根据图形中红色数字指定的长度，画出下面图形。边长为30。

循环7次，每次画一个等边三角形齿：
- 右转30°前进30（斜上）
- 右转120°前进30（斜下）
- 左转150°（回到水平方向）

## 要求

- 边长30，无颜色设置
- `p.hide()` 隐藏箭头
- 循环7次

## 期望输出

见原题图（锯齿状折线，7个齿）

## 难度

入门

## 标签

`循环` `折线` `角度` `锯齿` `三角形齿`

## 参考答案

```cpp
int main() {
    p.hide();
    for (int i = 0; i < 7; i++) {
        p.rt(30).fd(30);
        p.rt(120).fd(30);
        p.lt(150);
    }
    return 0;
}
```
