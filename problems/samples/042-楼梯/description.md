# 042 - 楼梯（魔法学院第6课）

**来源**：https://noi.hnai.net/problem/310.html（椰程信奥，魔法学院第6课 P310）

## 题目描述

请编写程序，画出下面图形。边长30，颜色号11。

嵌套循环画楼梯：
- 外层循环5次（每次画一级台阶组）
- 内层循环6次：每次 `fd(30).rt(90)`（画正方形的6条边/1.5圈）
- 外层每次结束后 `p.rt(180)`（反向，准备画下一级）

## 要求

- 颜色11，隐藏箭头
- 外层循环 i=0..4，内层循环 j=0..5
- 内层 `p.fd(30).rt(90)`，外层末尾 `p.rt(180)`

## 期望输出

见原题图（阶梯状图案）

## 难度

进阶

## 标签

`嵌套循环` `颜色` `楼梯` `旋转` `复合图形`

## 参考答案

```cpp
int main() {
    pen.c(11).hide();
    for (int i = 0; i < 5; i++) {
        for (int j = 0; j < 6; j++)
            p.fd(30).rt(90);
        p.rt(180);
    }
    return 0;
}
```
