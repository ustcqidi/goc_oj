# 044 - 正十二边形（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

编程，画出正十二边形，边长50，颜色3。

## 要求

- `int n = 12`，颜色3，循环n次
- 每次 `pen.fd(50).rt(360.0/n)`

## 期望输出

见原题图（正十二边形，边长50，绿色）

## 难度

入门

## 标签

`循环` `正多边形` `浮点角度` `变量n` `颜色`

## 参考答案

```cpp
int main() {
    int n = 12;
    pen.c(3);
    for (int i = 0; i < n; i++) {
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```
