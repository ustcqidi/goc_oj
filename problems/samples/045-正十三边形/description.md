# 045 - 正十三边形（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

请编写程序，画出正十三边形，边长50，颜色4号。

## 要求

- `int n = 13`，颜色4，循环n次
- 每次 `pen.fd(50).rt(360.0/n)`

## 期望输出

见原题图（正十三边形，边长50，黄色）

## 难度

入门

## 标签

`循环` `正多边形` `浮点角度` `非整除角度`

## 参考答案

```cpp
int main() {
    int n = 13;
    p.hide();
    pen.c(4);
    for (int i = 0; i < n; i++) {
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```

> ✅ `360.0/13` 不是整数，必须用浮点除法。纯标准 GOC 命令，完全兼容 OJ 执行器！
