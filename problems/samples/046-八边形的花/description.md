# 046 - 八边形的花（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

编程，画出八边形的花，边长50，颜色3。

外层循环8次，每次：
- 内层循环3次画等边三角形：`pen.fd(50).lt(120)`
- 然后 `pen.fd(50).rt(360.0/8)` 前进并旋转到下一方向

## 要求

- 外层 n=8，内层3次（三角形）
- 颜色3，隐藏箭头
- 外层末尾 `pen.fd(50).rt(360.0/n)` 平移到下一角

## 期望输出

见原题图（8朵三角形花瓣均匀分布）

## 难度

进阶

## 标签

`嵌套循环` `正多边形` `花形` `等分旋转` `颜色`

## 参考答案

```cpp
int main() {
    int n = 8;
    p.hide();
    pen.c(3);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < 3; j++)
            pen.fd(50).lt(120);
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```
