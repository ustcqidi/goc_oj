# 047 - 六边形的花（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

编程，画出六边形的花，边长50，颜色11。

外层循环6次，每次：
- 内层循环4次画正方形：`pen.fd(50).rt(-90)`（即左转90°）
- 然后 `pen.fd(50).rt(360.0/6)` 前进并旋转

## 要求

- 外层 n=6，内层4次（正方形，注意 rt(-90) = lt(90)）
- 颜色11

## 期望输出

见原题图（6朵正方形花瓣均匀分布）

## 难度

进阶

## 标签

`嵌套循环` `正多边形` `花形` `等分旋转` `颜色` `负角度`

## 参考答案

```cpp
int main() {
    int n = 6;
    pen.c(11);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < 4; j++)
            pen.fd(50).rt(-90);
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```

> ✅ `rt(-90)` 等效于 `lt(90)`。纯标准 GOC 命令，完全兼容 OJ 执行器！
