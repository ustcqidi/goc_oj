# 048 - 空心十字型（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

编程，画出空心十字型，边长50，颜色14。

外层循环4次，每次：
- 内层循环3次画"n型"（3段连续）：`pen.fd(50).rt(90)`
- 然后 `pen.rt(180)` 旋转180°到下一臂

## 要求

- 外层4次，内层3次
- 颜色14
- 内层 rt(90)，外层末尾 rt(180)

## 期望输出

见原题图（空心十字，4臂均匀分布）

## 难度

进阶

## 标签

`嵌套循环` `折线` `十字型` `旋转180°`

## 参考答案

```cpp
int main() {
    pen.c(14);
    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 3; j++)
            pen.fd(50).rt(90);
        pen.rt(180);
    }
    return 0;
}
```

> ✅ 内层3次 rt(90) 共270°旋转，外层 rt(180) 再加180°，共旋转 4×(270°+180°) = 4×450° = 1800°。纯标准 GOC 命令，完全兼容 OJ 执行器！
