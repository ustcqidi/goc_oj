# 049 - 十心徽型（魔法学院第7课）

**来源**：https://noi.hnai.net/problem/311.html（椰程信奥，魔法学院第7课 P311）

## 题目描述

请编写程序，画出十心徽型，边长50，颜色10。

外层循环10次，每次：
- 内层循环3次画等边三角形：`pen.fd(50).rt(120)`
- 然后 `p.fd(50).rt(36)` 前进并旋转36°到下一臂

## 要求

- 外层10次，内层3次（等边三角形，rt(120)）
- 颜色10
- 外层末尾 `fd(50).rt(36)`（360°÷10=36°）

## 期望输出

见原题图（10朵三角形均匀分布成圆形徽章）

## 难度

进阶

## 标签

`嵌套循环` `正多边形` `花形` `等分360°` `颜色`

## 参考答案

```cpp
int main() {
    pen.c(10);
    for (int i = 0; i < 10; i++) {
        for (int j = 0; j < 3; j++)
            pen.fd(50).rt(120);
        p.fd(50);
        pen.rt(36);
    }
    return 0;
}
```

> ✅ `360°÷10=36°`，每次旋转36°让10个三角花瓣均匀排列。纯标准 GOC 命令，完全兼容 OJ 执行器！
