# 050 - 实心矩形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

请编写程序，画出实心矩形（100×80），颜色14。

## 要求

- `p.rr(100, 80, 14)` 画实心矩形
- 隐藏箭头

## 期望输出

见原题图（实心矩形，青色）

## 难度

入门

## 标签

`实心矩形` `rr命令` `颜色`

## 参考答案

```cpp
int main() {
    p.hide();
    p.rr(100, 80, 14);
    return 0;
}
```

> ⚠️ `p.rr(w,h,color)` 是扩展 GOC 命令（实心矩形），标准 OJ 执行器可能不支持。
