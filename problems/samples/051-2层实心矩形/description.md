# 051 - 2层实心矩形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

请编写程序，画出2层实心矩形：外层100×80颜色14，内层80×50颜色0（黑色）。

## 要求

- `p.rr(100, 80, 14)` 外层矩形
- `p.rr(80, 50, 0)` 内层矩形（覆盖中心）

## 期望输出

见原题图（双层矩形，外青内黑）

## 难度

入门

## 标签

`实心矩形` `rr命令` `颜色` `叠加`

## 参考答案

```cpp
int main() {
    p.hide();
    p.rr(100, 80, 14);
    p.rr(80, 50, 0);
    return 0;
}
```

> ⚠️ `p.rr()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
