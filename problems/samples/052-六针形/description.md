# 052 - 六针形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

请编程，画出6针形图案，边长100。6根针均匀放射，每次右转60°。

## 要求

- 循环6次，每次 `fd(100).bk(100).rt(60)`

## 期望输出

见原题图（6根均匀放射线，雪花状）

## 难度

入门

## 标签

`循环` `放射线` `抬笔落笔` `等分旋转`

## 参考答案

```cpp
int main() {
    for (int i = 0; i < 6; i++)
        p.fd(100).bk(100).rt(60);
    return 0;
}
```
