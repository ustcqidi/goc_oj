# 053 - 彩色15针形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

编程，画出彩色15针形，边长100。15根针均匀放射，每根颜色不同(0-14)。

## 要求

- `int n = 15`，循环n次
- 每次：设颜色 i，前进100，后退100，右转 `360.0/n`

## 期望输出

见原题图（15根彩色放射线）

## 难度

入门

## 标签

`循环` `放射线` `颜色` `浮点角度` `等分旋转`

## 参考答案

```cpp
int main() {
    int n = 15;
    for (int i = 0; i < n; i++) {
        pen.c(i);
        pen.fd(100).bk(100);
        pen.rt(360.0 / n);
    }
    return 0;
}
```
