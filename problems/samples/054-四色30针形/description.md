# 054 - 四色30针形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

编程，画出四色30针形，笔粗5，边长100。30根针均匀放射，颜色按 i%4 循环（0,1,2,3）。

## 要求

- `int n = 30`，`pen.size(5)`
- 每次：`pen.c(i%4)`，前进100，后退100，右转 `360.0/n`

## 期望输出

见原题图（30根4色放射线）

## 难度

入门

## 标签

`循环` `放射线` `颜色` `取余` `size` `浮点角度`

## 参考答案

```cpp
int main() {
    int n = 30;
    pen.size(5);
    for (int i = 0; i < n; i++) {
        pen.c(i % 4);
        pen.fd(100).bk(100);
        pen.rt(360.0 / n);
    }
    return 0;
}
```
