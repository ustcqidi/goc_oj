# 055 - 彩色环形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

画彩色环形，笔粗8，边长50。10条线段均匀形成多边形（正十边形），每段不同颜色。

## 要求

- `int n = 10`，`pen.size(8).hide()`
- 每次：设颜色 i，前进50，右转 `360.0/n`

## 期望输出

见原题图（彩色正十边形环）

## 难度

入门

## 标签

`循环` `正多边形` `颜色` `size` `浮点角度`

## 参考答案

```cpp
int main() {
    int n = 10;
    pen.size(8).hide();
    for (int i = 0; i < n; i++) {
        pen.c(i);
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```
