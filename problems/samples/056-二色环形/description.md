# 056 - 二色环形（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

编程，画出二色环形，笔粗8，边长50，颜色号2和3交替。10段正十边形，奇偶段颜色交替。

## 要求

- `int n = 10`，`pen.size(8).hide()`
- 每次：`pen.c(i%2+2)`（颜色2或3），前进50，右转 `360.0/n`

## 期望输出

见原题图（绿蓝交替的正十边形）

## 难度

入门

## 标签

`循环` `正多边形` `颜色` `取余` `交替` `size`

## 参考答案

```cpp
int main() {
    int n = 10;
    pen.size(8).hide();
    for (int i = 0; i < n; i++) {
        pen.c(i % 2 + 2);
        pen.fd(50);
        pen.rt(360.0 / n);
    }
    return 0;
}
```

> ✅ `i%2+2` 在2和3之间交替。纯标准 GOC 命令，完全兼容 OJ 执行器！
