# 057 - 彩色螺旋（魔法学院第8课）

**来源**：https://noi.hnai.net/problem/312.html（椰程信奥，魔法学院第8课 P312）

## 题目描述

编程，画出彩色螺旋，从里到外边长依次为10,20,30,...。共15段，每段颜色不同，固定右转90°。

## 要求

- `int n = 15`，循环n次
- 每次：设颜色 i，前进 `10+i*10`，右转90°

## 期望输出

见原题图（彩色方形螺旋，边长递增）

## 难度

入门

## 标签

`循环` `螺旋` `颜色` `变步长` `等差递增`

## 参考答案

```cpp
int main() {
    int n = 15;
    for (int i = 0; i < n; i++) {
        pen.c(i);
        pen.fd(10 + i * 10);
        pen.rt(90);
    }
    return 0;
}
```
