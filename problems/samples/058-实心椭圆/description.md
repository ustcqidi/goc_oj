# 058 - 实心椭圆（魔法学院第9课）

**来源**：https://noi.hnai.net/problem/313.html（椰程信奥，魔法学院第9课 P313）

## 题目描述

请编写程序，画出实心椭圆，长半径100，短半径50，颜色号3。

## 要求

- `p.ee(100, 50, 3)` 画实心椭圆

## 期望输出

见原题图（绿色实心椭圆）

## 难度

入门

## 标签

`实心椭圆` `ee命令` `颜色`

## 参考答案

```cpp
int main() {
    p.hide().ee(100, 50, 3);
    return 0;
}
```

> ⚠️ `p.ee(rx, ry, color)` 是扩展 GOC 命令（实心椭圆），标准 OJ 执行器可能不支持。
