# 059 - 双色椭圆（魔法学院第9课）

**来源**：https://noi.hnai.net/problem/313.html（椰程信奥，魔法学院第9课 P313）

## 题目描述

请编写程序，画出双色椭圆，外层100×50颜色0（黑），内层50×50颜色6（青）。

## 要求

- 外层：`p.ee(100, 50, 0)`
- 内层：`p.ee(50, 50, 6)`（正圆）

## 期望输出

见原题图（外黑内青的双层椭圆/圆形）

## 难度

入门

## 标签

`实心椭圆` `ee命令` `颜色` `叠加`

## 参考答案

```cpp
int main() {
    p.hide().ee(100, 50, 0);
    p.ee(50, 50, 6);
    return 0;
}
```

> ⚠️ `p.ee()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
