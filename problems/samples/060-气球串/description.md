# 060 - 气球串（魔法学院第9课）

**来源**：https://noi.hnai.net/problem/313.html（椰程信奥，魔法学院第9课 P313）

## 题目描述

编程，画出气球串。椭圆中心距离50，长半径分别是20/40/60/80/100，短半径是长半径一半。颜色号分别是1/2/3/4/5。

- 抬笔，循环 i 从1到5
- 每次：画椭圆 `p.ee(i*20, i*10, i)`，再 `p.fd(50)` 前进

## 要求

- `p.up().hide()`
- `for(i=1;i<6;i++) { p.ee(i*20,i*10,i); p.fd(50); }`

## 期望输出

见原题图（5个越来越大的彩色椭圆，串在一条线上）

## 难度

入门

## 标签

`循环` `实心椭圆` `ee命令` `颜色` `平移`

## 参考答案

```cpp
int main() {
    p.up().hide();
    for (int i = 1; i < 6; i++) {
        p.ee(i * 20, i * 10, i);
        p.fd(50);
    }
    return 0;
}
```

> ⚠️ `p.ee()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
