# 061 - 同心圆环（魔法学院第9课）

**来源**：https://noi.hnai.net/problem/313.html（椰程信奥，魔法学院第9课 P313）

## 题目描述

请编程，画出同心圆环。圆半径分别为20/40/60/80/100/120，颜色号分别为1/2/3/4/5/6。

逆序循环（i 从6到1），每次 `p.oo(i*20, i)` 画圆。大圆先画，小圆后画覆盖，形成同心圆。

## 要求

- `for(i=6; i>0; i--)` 逆序循环
- 每次 `p.oo(i*20, i)` 画半径递减、颜色递减的圆

## 期望输出

见原题图（6个同心圆，从外到内半径递减，颜色6→1）

## 难度

入门

## 标签

`循环` `逆序` `圆` `同心圆` `颜色`

## 参考答案

```cpp
int main() {
    for (int i = 6; i > 0; i--) {
        p.oo(i * 20, i);
    }
    return 0;
}
```
