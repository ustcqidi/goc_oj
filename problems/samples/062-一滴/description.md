# 062 - 一滴（魔法学院第9课）

**来源**：https://noi.hnai.net/problem/313.html（椰程信奥，魔法学院第9课 P313）

## 题目描述

编程，画出"一滴"图形。6个黑红相间的圆，圆心距离50，圆半径分别为180/150/120/90/60/30（从大到小）。

- 抬笔，逆序循环 i 从6到1
- 每次：`p.oo(i*30, i%2)` 画圆（颜色0和1交替），再 `p.fd(50)` 前进

## 要求

- `p.up().hide()`
- `for(i=6;i>0;i--) { p.oo(i*30, i%2); p.fd(50); }`

## 期望输出

见原题图（6个递减大小的黑红圆串）

## 难度

入门

## 标签

`循环` `逆序` `圆` `取余` `平移` `颜色`

## 参考答案

```cpp
int main() {
    p.up().hide();
    for (int i = 6; i > 0; i--) {
        p.oo(i * 30, i % 2);
        p.fd(50);
    }
    return 0;
}
```
