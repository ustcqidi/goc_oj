# 065 - 内切圆（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

请编程，画出内切圆，矩形边长100。矩形黑色(0)，内切圆红色(1)，半径50。

## 要求

- `pen.hide().rr(100, 100, 0)` 画黑色实心正方形
- `pen.oo(50, 1)` 画红色实心圆覆盖

## 期望输出

见原题图（黑色正方形内有红色圆）

## 难度

入门

## 标签

`实心矩形` `圆` `颜色` `叠加` `rr命令`

## 参考答案

```cpp
int main() {
    pen.hide().rr(100, 100, 0);
    pen.oo(50, 1);
    return 0;
}
```

> ⚠️ `pen.rr()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
