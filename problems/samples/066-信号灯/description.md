# 066 - 信号灯（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

编程，画出信号灯图形，矩形边长50×140，三个圆半径20（红/黄/绿）。

## 要求

- 黑色外框矩形 `rr(50, 140, 0)`
- 三个圆垂直排列：红(1)、黄(4)、绿(3)，各半径20

> ⚠️ 原页面代码为3色花环的代码（疑似复制错误），以下为推断重构。

## 期望输出

见原题图（交通信号灯）

## 难度

入门

## 标签

`实心矩形` `圆` `颜色` `rr命令` `组合图形`

## 参考答案

```cpp
int main() {
    pen.hide().rr(50, 140, 0);   // 外框
    pen.up().lt(90).fd(40).rt(90);  // 移到第一个圆位置
    pen.oo(20, 1);  // 红
    pen.fd(40);
    pen.oo(20, 4);  // 黄
    pen.fd(40);
    pen.oo(20, 3);  // 绿
    return 0;
}
```

> ⚠️ `pen.rr()` 是扩展 GOC 命令，以上为推断重构。
