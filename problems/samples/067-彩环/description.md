# 067 - 彩环（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

编程，画出彩环。笔粗10，边长30，四种颜色号分别是11、13、10、14，循环18次。

- `for(i=0;i<18;i++)` 循环18次
- 每次用 `i%4` 选颜色(0→11,1→13,2→10,3→14)，前进30，右转 `360.0/18=20°`

## 要求

- `pen.size(10).hide()`
- 18次循环，颜色按 i%4 分四段
- 实际效果：18边彩色多边形轮廓

## 期望输出

见原题图（18段4色多边形）

## 难度

入门

## 标签

`循环` `颜色` `取余` `if判断` `正多边形` `size`

## 参考答案

```cpp
int main() {
    pen.size(10).hide();
    for (int i = 0; i < 18; i++) {
        if (i % 4 == 0) pen.c(11);
        if (i % 4 == 1) pen.c(13);
        if (i % 4 == 2) pen.c(10);
        if (i % 4 == 3) pen.c(14);
        pen.fd(30).rt(360.0 / 18);
    }
    return 0;
}
```
