# 068 - 双色漏斗（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

编程，画出双色漏斗。颜色号1、2，线段长度5/10/15/20...200（步长5，共41段）。

- `for(i=0;i<=40;i++)` 41次循环
- 偶数用颜色2，奇数用颜色1
- 每次前进 `i*5` 步，右转90°

## 要求

- 循环 i 从0到40（含40，共41次）
- `if(i%2==0) c(2); if(i%2==1) c(1)`
- `fd(i*5).rt(90)`

## 期望输出

见原题图（双色方形螺旋漏斗）

## 难度

入门

## 标签

`循环` `螺旋` `颜色` `取余` `变步长` `双色`

## 参考答案

```cpp
int main() {
    p.hide().speed(10);
    for (int i = 0; i <= 40; i++) {
        if (i % 2 == 0) pen.c(2);
        if (i % 2 == 1) pen.c(1);
        pen.fd(i * 5).rt(90);
    }
    return 0;
}
```
