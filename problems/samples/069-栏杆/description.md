# 069 - 栏杆（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

编程，画出栏杆，2种颜色分别是10、14，矩形边长20，圆半径10。循环16次。

- 右转90°后朝右前进
- `i%3==0` 时画矩形，`i%3==1/2` 时画圆
- 每次前进20步

## 要求

- `pen.up().hide().rt(90)`
- `for(i=0;i<16;i++) { if(i%3==0) rr(20,20,10); if(i%3==1||i%3==2) oo(10,14); fd(20); }`

## 期望输出

见原题图（矩形+圆+圆周期排列的栏杆）

## 难度

进阶

## 标签

`循环` `条件` `取余` `圆` `rr命令` `组合图形`

## 参考答案

```cpp
int main() {
    pen.up().hide().rt(90);
    for (int i = 0; i < 16; i++) {
        if (i % 3 == 0) pen.rr(20, 20, 10);
        if (i % 3 == 1) pen.oo(10, 14);
        if (i % 3 == 2) pen.oo(10, 14);
        pen.fd(20);
    }
    return 0;
}
```

> ⚠️ `pen.rr()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
