# 070 - 三色花环（魔法学院第10课）

**来源**：https://noi.hnai.net/problem/314.html（椰程信奥，魔法学院第10课 P314）

## 题目描述

编程，画出三色花环。圆心到中心的距离100，圆半径20，3种颜色号为13、6、10。循环12次，每次前进100画圆再回来，右转30°。

## 要求

- `pen.up()`，循环12次
- 每次：`fd(100)`，按 `i%3` 选颜色(0→13,1→6,2→10) 画圆 `o(20, color)`，`bk(100).rt(360.0/12)`

## 期望输出

见原题图（12个圆点围成花环，3种颜色循环）

## 难度

入门

## 标签

`循环` `颜色` `取余` `圆` `放射状圆点`

## 参考答案

```cpp
int main() {
    pen.up();
    for (int i = 0; i < 12; i++) {
        p.fd(100);
        if (i % 3 == 0) pen.o(20, 13);
        if (i % 3 == 1) pen.o(20, 6);
        if (i % 3 == 2) pen.o(20, 10);
        p.bk(100).rt(360.0 / 12);
    }
    return 0;
}
```

> ✅ 纯标准 GOC 命令，完全兼容 OJ 执行器！（`pen.o()` 为空心圆命令）
