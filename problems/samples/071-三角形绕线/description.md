# 071 - 三角形绕线（魔法学院第11课）

**来源**：https://noi.hnai.net/problem/315.html（椰程信奥，魔法学院第11课 P315）

## 题目描述

编程，画出三角形绕线图形。线段长度分别为10,20,30,...400（共40段），固定右转120°。

## 要求

- `for(i=1;i<=40;i++) { pen.fd(i*10); pen.rt(120); }`
- 三角形的外角120°，步长等差递增

## 期望输出

见原题图（螺旋三角形，从小到大）

## 难度

入门

## 标签

`循环` `螺旋` `三角形` `变步长` `等差递增`

## 参考答案

```cpp
int main() {
    p.hide().speed(10);
    for (int i = 1; i <= 40; i++) {
        pen.fd(i * 10);
        pen.rt(120);
    }
    return 0;
}
```
