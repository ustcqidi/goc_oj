# 072 - 选择角度（填空题，魔法学院第11课）

**来源**：https://noi.hnai.net/problem/315.html（椰程信奥，魔法学院第11课 P315）

## 题目描述

有程序：前进 i 步，右转____度，循环100次。选哪个角度能画出漂亮花形？

选项：A.203 B.91 C.103 D.95

**答案：B（91°）**

91不是360的因数，`100×91 = 9100 = 25×360 + 100`，螺旋缓慢旋转，形成螺旋花。203、103、95同理，选91效果最佳（接近正方形但每次偏转1°）。

## 要求

- 填空题：理解非整除角度产生的视觉效果
- 91° ≈ 90°+1°，每次旋转比正方形多1°，100次后共多100°，产生漂亮花形
