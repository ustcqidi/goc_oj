# 073 - 选择颜色（双色非整角螺旋，魔法学院第11课）

**来源**：https://noi.hnai.net/problem/315.html（椰程信奥，魔法学院第11课 P315）

## 题目描述

编程，画出双色非整角螺旋。线粗3，步长从1到200，每次旋转91°，奇偶段交替颜色3和5。

## 要求

- `pen.size(3)`，循环 i 从1到200
- `if(i%2==0) c(3); if(i%2==1) c(5);`
- `pen.fd(i).rt(91);`

## 期望输出

见原题图（双色非整角螺旋花）

## 难度

入门

## 标签

`循环` `螺旋` `颜色` `取余` `非整除角度` `size`

## 参考答案

```cpp
int main() {
    p.hide().speed(10).size(3);
    for (int i = 1; i <= 200; i++) {
        if (i % 2 == 0) pen.c(3);
        if (i % 2 == 1) pen.c(5);
        pen.fd(i);
        pen.rt(91);
    }
    return 0;
}
```
