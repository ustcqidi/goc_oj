# 074 - 选择颜色（填空题，魔法学院第11课）

**来源**：https://noi.hnai.net/problem/315.html（椰程信奥，魔法学院第11课 P315）

## 题目描述

有程序：步长i，旋转91°，颜色填`____`，循环200次。选哪个颜色表达式？

选项：A.3 B.4 C.i%2 D.1+i%2 E.i%8 F.1+i%9 G.6+i%6

**分析：**
- A/B：单色，无变化
- C：i%2 = 0或1（颜色0黑或1红）
- D：1+i%2 = 1或2（两色交替）
- E：i%8 = 0-7（8色循环）
- F：1+i%9 = 1-9+...
- G：6+i%6 = 6-11（6种颜色循环）

**答案：D、E、G 均产生彩色效果，D最简单，G产生6色效果最丰富。**

## 参考答案（填空）

选 **G. 6+i%6**（或D/E，根据目标颜色效果）
