# 075/076/077 - 选择长度/角度（填空题，魔法学院第11课）

**来源**：https://noi.hnai.net/problem/315.html（椰程信奥，魔法学院第11课 P315）

## 题目描述

P315中剩余填空题：

**76.选择长度：** `fd(50); bk(___); rt(36);` 循环10次，选项：A.0 B.20 C.50 D.75
- 若 bk=50（C），每次回到原点，画10根均匀放射线（36°间距）✅
- 若 bk=0，不后退，向前螺旋

**77.选择旋转角度：** 三色螺旋，选角度：A.80 B.120 C.144 D.150 E.179
- 179≈180-1，接近半圈，螺旋最密集漂亮

**78.选择长度和角度：** 外层 `pen.o(50)` 画空心圆，`fd(100).rt(angle)` 循环50次，选角度：A.71 B.79 C.150 D.179

## 难度

入门

## 标签

`填空题` `角度` `长度` `理解程序` `非整除螺旋`

## 参考答案

- 76: **C（bk=50，放射线）**
- 77: **E（179°，最密集螺旋）**
- 78: **D（179°）**
