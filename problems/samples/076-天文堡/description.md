# 076 - 天文堡（画半圆）（魔法学院第12课）

**来源**：https://noi.hnai.net/problem/316.html（椰程信奥，魔法学院第12课 P316）

## 题目描述

编程，画出天文堡图形。半圆是红色（颜色1），黑色矩形边长100×50。

- 先画红色半圆（`p.oo(50,1)` 画全圆，矩形覆盖下半部）
- 后退25步，画黑色矩形 `p.rr(100, 25, 0)`（覆盖下半部）

## 要求

- `p.up().hide()`
- `p.oo(50, 1)` — 全圆（半径50）
- `p.bk(25)` — 后退到矩形起始位置
- `p.rr(100, 25, 0)` — 黑色矩形覆盖下半圆

## 期望输出

见原题图（红色半圆+黑色底座）

## 难度

入门

## 标签

`实心圆` `实心矩形` `rr命令` `颜色` `覆盖技巧`

## 参考答案

```cpp
int main() {
    p.up().hide();
    p.oo(50, 1);
    p.bk(25);
    p.rr(100, 25, 0);
    return 0;
}
```

> ⚠️ `p.rr()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
