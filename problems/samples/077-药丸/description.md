# 077 - 药丸（魔法学院第12课）

**来源**：https://noi.hnai.net/problem/316.html（椰程信奥，魔法学院第12课 P316）

## 题目描述

编程，画出药丸图形。2个半圆颜色3（绿色），中间颜色5（紫色）正方形边长100。

- `p.oo(50)` 画左半圆（绿色），`bk(100)` 后退100，再画右半圆 `p.oo(50)`
- 前进50步到中心，`p.rr(100,100,5)` 画绿色正方形覆盖中间

## 要求

- `p.up().hide().c(3)` — 绿色
- `p.oo(50).bk(100).oo(50)` — 两端半圆
- `p.fd(50)` — 回到中心
- `p.rr(100, 100, 5)` — 中间正方形

## 期望输出

见原题图（胶囊/药丸形状）

## 难度

入门

## 标签

`实心圆` `实心矩形` `rr命令` `颜色` `覆盖技巧` `组合图形`

## 参考答案

```cpp
int main() {
    p.up().hide().c(3);
    p.oo(50).bk(100).oo(50);
    p.fd(50);
    p.rr(100, 100, 5);
    return 0;
}
```

> ⚠️ `p.rr()` 是扩展 GOC 命令，标准 OJ 执行器可能不支持。
