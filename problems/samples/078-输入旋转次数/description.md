# 078 - 输入旋转次数（魔法学院第13课）

**来源**：https://noi.hnai.net/problem/317.html（椰程信奥，魔法学院第13课 P317）

## 题目描述

请编写程序，根据输入的旋转次数n，画出圆盘图形。例如输入32，画32个圆点均匀排列在半径100的圆周上，颜色按 i%4 循环。

输入：一个正整数n（旋转次数）
输出：对应圆盘图形

## 要求

- `cin >> n` 读入旋转次数
- 循环n次：设颜色 i%4，前进100画圆(oo(15))后退，右转 `360.0/n`

## 期望输出

见原题图（n个圆点均匀分布，4色循环）

## 难度

进阶

## 标签

`cin输入` `循环` `圆点` `颜色` `取余` `参数化`

## 参考答案

```cpp
int main() {
    int n;
    cin >> n;
    p.hide().speed(10);
    for (int i = 0; i < n; i++) {
        p.c(i % 4);
        p.fd(100).oo(15).bk(100);
        p.rt(360.0 / n);
    }
    return 0;
}
```
