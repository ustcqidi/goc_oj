# 079 - 输入颜色（魔法学院第13课）

**来源**：https://noi.hnai.net/problem/317.html（椰程信奥，魔法学院第13课 P317）

## 题目描述

请编程，运行后根据输入的数值，画出该颜色号的半径为100的圆。

输入：一个整数（颜色号）

## 要求

- `cin >> a` 读入颜色
- `pen.oo(100, a)` 画对应颜色的圆

## 难度

入门

## 标签

`cin输入` `圆` `颜色` `参数化`

## 参考答案

```cpp
int main() {
    int a;
    cin >> a;
    pen.hide().oo(100, a);
    return 0;
}
```

> ✅ 纯标准 GOC 命令（含 cin 输入），完全兼容 OJ 执行器！
