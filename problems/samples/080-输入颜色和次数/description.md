# 080 - 输入颜色和旋转次数（魔法学院第13课）

**来源**：https://noi.hnai.net/problem/317.html（椰程信奥，魔法学院第13课 P317）

## 题目描述

请编写程序，根据输入的颜色和旋转次数，画出对应颜色的三角形花形。例如输入 3 8，用3号颜色画8个三角形，边长100。

输入：两个整数（颜色号 次数）

## 参考答案

int main() {
    int a, n;
    cin >> a >> n;
    p.hide().c(a);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < 3; j++)
            pen.fd(100).lt(120);
        pen.rt(360.0 / n);
    }
    return 0;
}
