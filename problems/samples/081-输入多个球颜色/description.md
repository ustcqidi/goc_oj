# 081 - 输入多个球颜色（魔法学院第13课）

**来源**：https://noi.hnai.net/problem/317.html（椰程信奥，魔法学院第13课 P317）

## 题目描述

请编写程序，根据输入的旋转次数和每个球的颜色，画出圆环图形。圆半径30，圆心距中心100。

输入格式：第一行正整数n；第二行n个颜色号。

## 要求

- `cin >> n`，再循环n次 `cin >> a`
- 每次：`p.fd(100).oo(30, a)` — 前进100画圆，再 `p.rt(360.0/n)`

## 期望输出

见原题图（n个不同颜色圆均匀分布在圆周上）

## 难度

进阶

## 标签

`cin输入` `循环读入` `圆点` `颜色` `放射状`

## 参考答案

```cpp
int main() {
    int n, a;
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> a;
        p.fd(100).oo(30, a);
        p.rt(360.0 / n);
    }
    return 0;
}
```

> ✅ 注意：每次循环不返回中心（没有 bk），圆点沿旋转路径分布。含 cin 输入，兼容 OJ 执行器。
