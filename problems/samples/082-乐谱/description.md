# 082 - 乐谱（魔法学院第14课）

**来源**：https://noi.hnai.net/problem/318.html（椰程信奥，魔法学院第14课 P318）

## 题目描述

输入n个音符数值，画n个矩形音符条。矩形宽10，颜色由输入确定，高度是颜色号×30，相邻矩形中心距15。

输入：第一行n；第二行n个颜色号

## 要求

- `cin >> n`，循环读 a，`pen.rr(10, a*30, a)` 画矩形
- `p.rt(90).fd(15).lt(90)` 移到下一位置

## 难度

进阶

## 标签

`cin输入` `rr命令` `颜色` `柱状图` `参数化`

## 参考答案

```cpp
int main() {
    int n, a;
    cin >> n;
    p.hide().up();
    for (int i = 0; i < n; i++) {
        cin >> a;
        pen.rr(10, a * 30, a);
        p.rt(90).fd(15).lt(90);
    }
    return 0;
}
```

> ⚠️ `pen.rr()` 是扩展命令；含 cin 输入。
