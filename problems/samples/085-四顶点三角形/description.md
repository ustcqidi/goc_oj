# 085 - 四个顶点三角形（程序填空，2020南海区比赛）

**来源**：https://noi.hnai.net/problem/294.html（椰程信奥，2020年南海区小学乙组比赛 P294）

## 题目描述

在四边形的四个顶点上，画完全对称的四个三角形。每条边的边长为100。

外层循环4次（正方形4条边），每次：
- 前进100步到下一顶点
- 左转75°
- 内层循环3次画等边三角形（fd(100).rt(120)）
- 右转165°（复位方向）

## 要求

- 外层4次，内层3次
- 外层：`p.fd(100).lt(75)` + 内层三角形 + `p.rt(165)`

## 期望输出

见原题图（正方形四个角各有一个等边三角形）

## 难度

进阶

## 标签

`嵌套循环` `三角形` `正方形` `角度计算` `对称图形`

## 参考答案

```cpp
int main() {
    for (int i = 0; i < 4; i++) {
        p.fd(100);
        p.lt(75);
        for (int j = 0; j < 3; j++)
            p.fd(100).rt(120);
        p.rt(165);
    }
    p.hide();
    return 0;
}
```
