# 086 - 彩色虚线（2020南海区比赛）

**来源**：https://noi.hnai.net/problem/294.html（椰程信奥，2020年南海区小学乙组比赛 P294）

## 题目描述

用0、1、2号颜色，画彩色虚线。线段长度和空的间距长度都是20。

- 右转90°朝右
- 循环9次：`p.c(i%3).fd(20).up().fd(20).down()`

## 要求

- `p.rt(90)` 朝右
- 循环9次，颜色 i%3（0,1,2循环）
- 每次：画20步，抬笔20步空白，落笔

## 期望输出

见原题图（彩色虚线，3色循环）

## 难度

入门

## 标签

`循环` `颜色` `取余` `抬笔落笔` `虚线`

## 参考答案

```cpp
int main() {
    p.rt(90);
    for (int i = 0; i < 9; i++) {
        p.c(i % 3).fd(20).up().fd(20).down();
    }
    return 0;
}
```
