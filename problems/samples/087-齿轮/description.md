# 087 - 齿轮（2020南海区比赛）

**来源**：https://noi.hnai.net/problem/294.html（椰程信奥，2020年南海区小学乙组比赛 P294）

## 题目描述

键盘输入一个正整数n，画n个齿的齿轮。内圆半径80，每个齿由边长20和40的实心矩形构成。

## 要求

- `p.o(80)` 画内圆轮廓
- `p.picU(0)` — 扩展命令（切换绘图模式）
- 循环n次：前进100，`rr(20,40)` 画矩形齿，后退100，右转 `360.0/n`

## 难度

进阶

## 标签

`cin输入` `循环` `rr命令` `齿轮` `picU命令`

## 参考答案

```cpp
int main() {
    p.o(80);
    p.up();
    p.picU(0);
    int n;
    cin >> n;
    for (int i = 0; i < n; i++)
        p.fd(100).rr(20, 40).bk(100).rt(360.0 / n);
    return 0;
}
```

> ⚠️ `p.picU()` 和 `p.rr()` 是扩展命令，标准 OJ 不支持。
