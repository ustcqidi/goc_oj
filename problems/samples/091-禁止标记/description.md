# 091 - 禁止标记（stop，2018NHOI小乙）

**来源**：https://noi.hnai.net/problem/295.html（椰程信奥，2018年南海区小学乙组比赛 P295）

## 题目描述

用红色线粗10画禁止标记：圆半径100，2条直径相互垂直并且左右对称（斜45°）。

## 要求

- `p.c(1).size(10).o(100)` 画红色圆
- 右转45°后，4次 `fd(100).bk(100).rt(45)` 画斜线

## 期望输出

见原题图（圆+斜十字线）

## 难度

入门

## 标签

`圆` `放射线` `颜色` `size` `角度`

## 参考答案

```cpp
int main() {
    p.c(1).size(10).o(100);
    p.rt(45).fd(100).bk(100).rt(45);
    p.rt(45).fd(100).bk(100).rt(45);
    p.rt(45).fd(100).bk(100).rt(45);
    p.rt(45).fd(100).bk(100).rt(45);
    return 0;
}
```
