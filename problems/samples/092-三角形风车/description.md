# 092 - 三角形风车（traingle，2018NHOI小乙）

**来源**：https://noi.hnai.net/problem/295.html（椰程信奥，2018年南海区小学乙组比赛 P295）

## 题目描述

画彩色三角形风车，每条边长均是100，颜色编号分别是1、2、3。3个三角形均匀旋转排列。

## 要求

方法一（双循环）：外层3次，内层3次画三角形，外层末尾 `rt(120)`。
方法二（单循环）：9次，颜色 `(j-1)%3+1`，每3次 `rt(120)`。

## 期望输出

见原题图（3色3角风车）

## 难度

进阶

## 标签

`嵌套循环` `三角形` `颜色` `旋转` `风车`

## 参考答案（方法一）

```cpp
int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
            p.c(j).fd(100).rt(120);
        }
        p.rt(120);
    }
    return 0;
}
```
