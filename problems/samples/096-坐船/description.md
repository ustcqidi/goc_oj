# 096 - 坐船（board，2018NHOI小乙）

**来源**：https://noi.hnai.net/problem/295.html（椰程信奥，2018年南海区小学乙组比赛 P295）

## 题目描述

输入N人体重，每船最多2人/100kg，最多10艘船。用矩形+圆画出运送方案（先画1人船再画2人船）。

## 难度

高级

## 标签

`cinWin` `数组` `排序` `moveTo` `r命令` `oo命令` `算法`

## 参考答案

见原题解（贪心算法+双指针配对+冒泡排序）。⚠️ 使用 cinWin, moveTo, r() 等扩展命令。绘图部分用 oo() 标准命令。
